\section*{Problem 7}
Ako su $x_1$ i $x_2$ koreni kvadratne jednaƒçine $\frac{1}{x - 1} + \frac{1}{x - 2} = 1$, tada je izraz $\frac{x_1}{x_2} + \frac{x_2}{x_1}$ jednak:

(A) $\frac{1}{3}$ \quad (B) $\frac{2}{3}$ \quad (C) $2$ \quad (D) $3$ \quad (E) $5$ \quad (N) Ne znam

\subsection*{Solution}
First, let's solve the equation $\frac{1}{x - 1} + \frac{1}{x - 2} = 1$.

Finding a common denominator:
$$\frac{x - 2 + x - 1}{(x - 1)(x - 2)} = 1$$
$$\frac{2x - 3}{(x - 1)(x - 2)} = 1$$

Cross-multiplying:
$$2x - 3 = (x - 1)(x - 2)$$
$$2x - 3 = x^2 - 3x + 2$$
$$0 = x^2 - 3x + 2 - 2x + 3$$
$$0 = x^2 - 5x + 5$$

So we have the quadratic equation $x^2 - 5x + 5 = 0$.

Using Vieta's formulas for $x^2 - 5x + 5 = 0$:
- Sum of roots: $x_1 + x_2 = 5$
- Product of roots: $x_1 \cdot x_2 = 5$

Now we need to find $\frac{x_1}{x_2} + \frac{x_2}{x_1}$:

$$\frac{x_1}{x_2} + \frac{x_2}{x_1} = \frac{x_1^2 + x_2^2}{x_1 x_2}$$

We need to find $x_1^2 + x_2^2$. Using the identity:
$$x_1^2 + x_2^2 = (x_1 + x_2)^2 - 2x_1x_2$$
$$x_1^2 + x_2^2 = 5^2 - 2 \cdot 5 = 25 - 10 = 15$$

Therefore:
$$\frac{x_1}{x_2} + \frac{x_2}{x_1} = \frac{x_1^2 + x_2^2}{x_1 x_2} = \frac{15}{5} = 3$$

\subsection*{Answer}
(D) $3$

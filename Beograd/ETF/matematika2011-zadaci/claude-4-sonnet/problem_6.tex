\section*{Problem 6}
Osnovice jednakokrakog trapeza su 15cm i 5cm a kraci 13cm. Njegova visina (u cm) iznosi:

(A) $16$ \quad (B) $8$ \quad (C) $10$ \quad (D) $12$ \quad (E) $9$ \quad (N) Ne znam

\subsection*{Solution}
Let's set up the problem. We have an isosceles trapezoid with:
- Longer base: $a = 15$ cm
- Shorter base: $b = 5$ cm  
- Equal legs: $c = 13$ cm
- Height: $h = ?$

In an isosceles trapezoid, if we drop perpendiculars from the ends of the shorter base to the longer base, we create two congruent right triangles on the sides and a rectangle in the middle.

The difference in base lengths is $15 - 5 = 10$ cm, so each right triangle has a base of $\frac{10}{2} = 5$ cm.

Using the Pythagorean theorem for one of these right triangles:
- Hypotenuse = leg of trapezoid = $13$ cm
- Base = $5$ cm  
- Height = $h$ cm

$$h^2 + 5^2 = 13^2$$
$$h^2 + 25 = 169$$
$$h^2 = 144$$
$$h = 12 \text{ cm}$$

\subsection*{Answer}
(D) $12$

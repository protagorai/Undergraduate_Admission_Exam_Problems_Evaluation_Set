\section*{Problem 10}
The lines $-ax + y - 3 = 0$ and $x - by + 2 = 0$ intersect at the center of the circle $x^2 + y^2 - 2x + 4y - 10 = 0$. Find the angle between these lines.

\subsection*{Solution}
First, find the center of the circle. Rewrite the circle equation in standard form:
\[
x^2 - 2x + y^2 + 4y = 10
\]
\[
(x^2 - 2x + 1) + (y^2 + 4y + 4) = 10 + 1 + 4
\]
\[
(x - 1)^2 + (y + 2)^2 = 15
\]

The center is $(1, -2)$.

Since both lines pass through $(1, -2)$:

For line 1: $-a(1) + (-2) - 3 = 0 \Rightarrow -a - 5 = 0 \Rightarrow a = -5$

For line 2: $1 - b(-2) + 2 = 0 \Rightarrow 1 + 2b + 2 = 0 \Rightarrow b = -\frac{3}{2}$

Now write the lines in slope-intercept form:

Line 1: $-ax + y - 3 = 0 \Rightarrow 5x + y - 3 = 0 \Rightarrow y = -5x + 3$, slope $m_1 = -5$

Line 2: $x - by + 2 = 0 \Rightarrow x + \frac{3}{2}y + 2 = 0 \Rightarrow y = -\frac{2}{3}x - \frac{4}{3}$, slope $m_2 = -\frac{2}{3}$

The angle $\theta$ between two lines with slopes $m_1$ and $m_2$ is:
\[
\tan\theta = \left|\frac{m_1 - m_2}{1 + m_1 m_2}\right| = \left|\frac{-5 - (-\frac{2}{3})}{1 + (-5)(-\frac{2}{3})}\right| = \left|\frac{-5 + \frac{2}{3}}{1 + \frac{10}{3}}\right|
\]
\[
= \left|\frac{-\frac{13}{3}}{\frac{13}{3}}\right| = |-1| = 1
\]

Therefore $\theta = 45°$.

\subsection*{Answer}
\textbf{(D)} $45°$

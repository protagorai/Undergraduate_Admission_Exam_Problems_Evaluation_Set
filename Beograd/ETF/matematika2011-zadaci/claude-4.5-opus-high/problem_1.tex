\section*{Problem 1}
Which of the five given expressions has a different value from the other four?
\[
\text{(A) } 2^8 \quad \text{(B) } 4^4 \quad \text{(C) } 8^{8/3} \quad \text{(D) } 16^2 \quad \text{(E) } 32^{6/5}
\]

\subsection*{Solution}
Let's calculate each expression:

\textbf{(A)} $2^8 = 256$

\textbf{(B)} $4^4 = (2^2)^4 = 2^8 = 256$

\textbf{(C)} $8^{8/3} = (2^3)^{8/3} = 2^8 = 256$

\textbf{(D)} $16^2 = (2^4)^2 = 2^8 = 256$

\textbf{(E)} $32^{6/5} = (2^5)^{6/5} = 2^6 = 64$

All expressions except (E) equal 256. Expression (E) equals 64.

\subsection*{Answer}
\textbf{(E)} $32^{6/5} = 64$

\section*{Problem 12}
Zbir beskonačne geometrijske progresije $\frac{\sqrt{2}+1}{\sqrt{2}-1} + \frac{1}{2-\sqrt{2}} + \frac{1}{2} + \dots$ jednak je:
\[
(A)\; 2+\sqrt{2} \quad (B)\; 2-\sqrt{2} \quad (C)\; 1 \quad (D)\; 4+3\sqrt{2} \quad (E)\; 4-3\sqrt{2}
\]

\subsection*{Solution}
Odredimo prva dva člana niza.
Prvi član:
\[
a_1 = \frac{\sqrt{2}+1}{\sqrt{2}-1} = \frac{(\sqrt{2}+1)^2}{(\sqrt{2}-1)(\sqrt{2}+1)} = \frac{2+2\sqrt{2}+1}{2-1} = 3+2\sqrt{2}
\]
Drugi član:
\[
a_2 = \frac{1}{2-\sqrt{2}} = \frac{2+\sqrt{2}}{4-2} = \frac{2+\sqrt{2}}{2} = 1+\frac{\sqrt{2}}{2}
\]
Treći član je $a_3 = \frac{1}{2}$.
Proverimo količnik $q$:
\[
q = \frac{a_3}{a_2} = \frac{1/2}{(2+\sqrt{2})/2} = \frac{1}{2+\sqrt{2}} \cdot \frac{2-\sqrt{2}}{2-\sqrt{2}} = \frac{2-\sqrt{2}}{4-2} = \frac{2-\sqrt{2}}{2} = 1-\frac{\sqrt{2}}{2}
\]
Proverimo da li je $a_2 / a_1$ isto:
\[
\frac{a_2}{a_1} = \frac{\frac{2+\sqrt{2}}{2}}{3+2\sqrt{2}} = \frac{2+\sqrt{2}}{2(3+2\sqrt{2})} \cdot \frac{3-2\sqrt{2}}{3-2\sqrt{2}} = \frac{6-4\sqrt{2}+3\sqrt{2}-4}{2(9-8)} = \frac{2-\sqrt{2}}{2}
\]
Količnik je $q = \frac{2-\sqrt{2}}{2}$. Kako je $|q| < 1$, suma beskonačnog reda je $S = \frac{a_1}{1-q}$.
\[
S = \frac{3+2\sqrt{2}}{1 - \frac{2-\sqrt{2}}{2}} = \frac{3+2\sqrt{2}}{\frac{2 - (2-\sqrt{2})}{2}} = \frac{2(3+2\sqrt{2})}{\sqrt{2}}
\]
\[
S = \sqrt{2}(3+2\sqrt{2}) = 3\sqrt{2} + 2(2) = 3\sqrt{2} + 4
\]

\subsection*{Answer}
$4+3\sqrt{2}$ (option \textbf{D}).

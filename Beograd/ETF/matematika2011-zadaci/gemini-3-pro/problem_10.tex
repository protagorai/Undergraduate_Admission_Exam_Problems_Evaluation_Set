\section*{Problem 10}
Prave $-ax+y-3=0$, $x-by+2=0$ seku se u centru kruga $x^2+y^2-2x+4y-10=0$. Ugao između ovih pravih je:
\[
(A)\; 60^\circ \quad (B)\; 30^\circ \quad (C)\; 90^\circ \quad (D)\; 45^\circ \quad (E)\; 75^\circ
\]

\subsection*{Solution}
Odredimo centar kruga. Jednačinu kruga svodimo na kanonski oblik:
\[
x^2 - 2x + y^2 + 4y = 10
\]
\[
(x-1)^2 - 1 + (y+2)^2 - 4 = 10
\]
\[
(x-1)^2 + (y+2)^2 = 15
\]
Centar kruga je $C(1, -2)$.
Obe prave prolaze kroz tačku $C(1, -2)$. Zamenimo koordinate centra u jednačine pravih da nađemo $a$ i $b$.
Prva prava: $-a(1) + (-2) - 3 = 0 \Rightarrow -a - 5 = 0 \Rightarrow a = -5$.
Jednačina prve prave je $5x + y - 3 = 0 \Rightarrow y = -5x + 3$. Koeficijent pravca je $k_1 = -5$.
Druga prava: $1 - b(-2) + 2 = 0 \Rightarrow 1 + 2b + 2 = 0 \Rightarrow 2b = -3 \Rightarrow b = -3/2$.
Jednačina druge prave je $x - (-3/2)y + 2 = 0 \Rightarrow x + \frac{3}{2}y + 2 = 0 \Rightarrow \frac{3}{2}y = -x - 2 \Rightarrow y = -\frac{2}{3}x - \frac{4}{3}$. Koeficijent pravca je $k_2 = -2/3$.

Tangens ugla $\varphi$ između pravih je:
\[
\tan \varphi = \left| \frac{k_2 - k_1}{1 + k_1 k_2} \right| = \left| \frac{-2/3 - (-5)}{1 + (-5)(-2/3)} \right| = \left| \frac{-2/3 + 15/3}{1 + 10/3} \right| = \left| \frac{13/3}{13/3} \right| = 1
\]
Kako je $\tan \varphi = 1$, sledi da je $\varphi = 45^\circ$.

\subsection*{Answer}
$45^\circ$ (option \textbf{D}).

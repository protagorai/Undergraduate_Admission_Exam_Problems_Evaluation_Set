\section*{Problem 8}
Kompleksan broj $\frac{\cos\alpha + i\sin\alpha + 1}{\cos\alpha + i\sin\alpha - 1}$ ($i=\sqrt{-1}, \alpha \ne 2k\pi, k \in \mathbf{Z}$), jednak je:
\[ (A)\ -i \cdot \text{ctg}\frac{\alpha}{2} \quad (B)\ -i \cdot \frac{2\sin\alpha}{1-\cos\alpha} \quad (C)\ -i \cdot \frac{2\sin\alpha}{2-\cos\alpha} \quad (D)\ -i \cdot \frac{\sin\alpha}{2(1-\cos\alpha)} \quad (E)\ -i \cdot \text{tg}\frac{\alpha}{2} \quad (N)\ \text{Ne znam} \]

\subsection*{Solution}
Let $z = \cos\alpha + i\sin\alpha$. We want to simplify $\frac{z+1}{z-1}$.
Using half-angle formulas:
$\cos\alpha = 2\cos^2\frac{\alpha}{2} - 1 = 1 - 2\sin^2\frac{\alpha}{2}$
$\sin\alpha = 2\sin\frac{\alpha}{2}\cos\frac{\alpha}{2}$
Numerator:
$\cos\alpha + 1 + i\sin\alpha = 2\cos^2\frac{\alpha}{2} + 2i\sin\frac{\alpha}{2}\cos\frac{\alpha}{2} = 2\cos\frac{\alpha}{2} (\cos\frac{\alpha}{2} + i\sin\frac{\alpha}{2})$.
Denominator:
$\cos\alpha - 1 + i\sin\alpha = -2\sin^2\frac{\alpha}{2} + 2i\sin\frac{\alpha}{2}\cos\frac{\alpha}{2} = 2i\sin\frac{\alpha}{2} (i\sin\frac{\alpha}{2} + \cos\frac{\alpha}{2})$.
Wait, check the factorization for denominator:
$-2\sin^2\frac{\alpha}{2} = 2i^2\sin^2\frac{\alpha}{2}$.
So $-2\sin^2\frac{\alpha}{2} + 2i\sin\frac{\alpha}{2}\cos\frac{\alpha}{2} = 2i\sin\frac{\alpha}{2} (i\sin\frac{\alpha}{2} + \cos\frac{\alpha}{2})$.
This matches.
The expression becomes:
\[ \frac{2\cos\frac{\alpha}{2} (\cos\frac{\alpha}{2} + i\sin\frac{\alpha}{2})}{2i\sin\frac{\alpha}{2} (\cos\frac{\alpha}{2} + i\sin\frac{\alpha}{2})} \]
Cancel common terms:
\[ \frac{\cos\frac{\alpha}{2}}{i\sin\frac{\alpha}{2}} = \frac{1}{i} \text{ctg}\frac{\alpha}{2} = -i \cdot \text{ctg}\frac{\alpha}{2} \]

\subsection*{Answer}
$-i \cdot \text{ctg}\frac{\alpha}{2}$ (option \textbf{A}).

\section*{Problem 9}
Ako je polinom $P(x) = x^{2014} + px^{2013} + qx - 1$ ($p, q \in \mathbb{R}$), deljiv polinomom $x + 1$, tada je zbir $p + q$ jednak:

\subsection*{Solution}
Ako je polinom $P(x)$ deljiv sa $x + 1$, tada je $P(-1) = 0$.

Računamo $P(-1)$:
\[
P(-1) = (-1)^{2014} + p(-1)^{2013} + q(-1) - 1
\]

Pošto je $2014$ paran broj, $(-1)^{2014} = 1$.
Pošto je $2013$ neparan broj, $(-1)^{2013} = -1$.

Dakle:
\[
P(-1) = 1 + p(-1) + q(-1) - 1 = 1 - p - q - 1 = -p - q
\]

Za $P(-1) = 0$:
\[
-p - q = 0
\]
\[
p + q = 0
\]

\subsection*{Answer}
$0$ (option \textbf{A}).

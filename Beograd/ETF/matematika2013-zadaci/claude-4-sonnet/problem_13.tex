\section*{Problem 13}
Ukupan broj realnih rešenja sistema jednačina $x^2 + xy - \sqrt{2} \cdot x = 0$, $x^2 + y^2 = 2$ je:

\subsection*{Solution}
Imamo sistem:
\begin{align}
x^2 + xy - \sqrt{2}x &= 0 \quad (1)\\
x^2 + y^2 &= 2 \quad (2)
\end{align}

Iz jednačine (1):
\[
x(x + y - \sqrt{2}) = 0
\]

Ovo daje $x = 0$ ili $x + y - \sqrt{2} = 0$.

\textbf{Slučaj 1:} $x = 0$

Iz jednačine (2): $0 + y^2 = 2$, pa je $y = \pm\sqrt{2}$.

Rešenja: $(0, \sqrt{2})$ i $(0, -\sqrt{2})$.

\textbf{Slučaj 2:} $x + y - \sqrt{2} = 0$, tj. $y = \sqrt{2} - x$

Zamenimo u jednačinu (2):
\[
x^2 + (\sqrt{2} - x)^2 = 2
\]
\[
x^2 + 2 - 2\sqrt{2}x + x^2 = 2
\]
\[
2x^2 - 2\sqrt{2}x = 0
\]
\[
2x(x - \sqrt{2}) = 0
\]

Ovo daje $x = 0$ ili $x = \sqrt{2}$.

Za $x = 0$: $y = \sqrt{2} - 0 = \sqrt{2}$ (već imamo).

Za $x = \sqrt{2}$: $y = \sqrt{2} - \sqrt{2} = 0$.

Rešenje: $(\sqrt{2}, 0)$.

Ukupno imamo 3 rešenja: $(0, \sqrt{2})$, $(0, -\sqrt{2})$, $(\sqrt{2}, 0)$.

\subsection*{Answer}
$3$ (option \textbf{E}).

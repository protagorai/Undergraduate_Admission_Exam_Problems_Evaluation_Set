\section*{Problem 12}
In the binomial expansion of $\left(\sqrt[3]{a} + \frac{1}{\sqrt[9]{a}}\right)^n$ where $a > 0$, $n \in \mathbb{N}$, the sum of the first three binomial coefficients is 121. Find the term that contains $\frac{1}{a}$.

\subsection*{Solution}
The first three binomial coefficients are $\binom{n}{0}$, $\binom{n}{1}$, $\binom{n}{2}$.

\[\binom{n}{0} + \binom{n}{1} + \binom{n}{2} = 1 + n + \frac{n(n-1)}{2} = 121\]

\[2 + 2n + n(n-1) = 242\]
\[2 + 2n + n^2 - n = 242\]
\[n^2 + n + 2 = 242\]
\[n^2 + n - 240 = 0\]
\[(n + 16)(n - 15) = 0\]

Since $n > 0$, we have $n = 15$.

The general term in the expansion is:
\[T_{k+1} = \binom{15}{k} \left(\sqrt[3]{a}\right)^{15-k} \left(\frac{1}{\sqrt[9]{a}}\right)^k = \binom{15}{k} a^{\frac{15-k}{3}} \cdot a^{-\frac{k}{9}}\]

\[= \binom{15}{k} a^{\frac{15-k}{3} - \frac{k}{9}} = \binom{15}{k} a^{\frac{3(15-k) - k}{9}} = \binom{15}{k} a^{\frac{45-3k-k}{9}} = \binom{15}{k} a^{\frac{45-4k}{9}}\]

For the term containing $\frac{1}{a} = a^{-1}$:
\[\frac{45-4k}{9} = -1\]
\[45 - 4k = -9\]
\[4k = 54\]
\[k = 13.5\]

This is not an integer, so there's no term with exactly $a^{-1}$.

\section*{Problem 2}
Find the number of real solutions of the equation $||1-|x||-1|-2=0$.

\subsection*{Solution}
We need to solve $||1-|x||-1|-2=0$, which means $||1-|x||-1|=2$.

This gives us two cases:
\begin{enumerate}
\item $|1-|x||-1 = 2 \Rightarrow |1-|x|| = 3$
\item $|1-|x||-1 = -2 \Rightarrow |1-|x|| = -1$ (impossible, absolute value cannot be negative)
\end{enumerate}

From case 1, $|1-|x|| = 3$:
\begin{itemize}
\item $1-|x| = 3 \Rightarrow |x| = -2$ (impossible)
\item $1-|x| = -3 \Rightarrow |x| = 4 \Rightarrow x = \pm 4$
\end{itemize}

So we have exactly 2 solutions: $x = 4$ and $x = -4$.

\subsection*{Verification}
For $x = 4$: $||1-4|-1|-2 = ||{-3}|-1|-2 = |3-1|-2 = 2-2 = 0$ \checkmark

For $x = -4$: $||1-|-4||-1|-2 = ||1-4|-1|-2 = |3-1|-2 = 0$ \checkmark

\subsection*{Answer}
$2$ (option \textbf{C}).

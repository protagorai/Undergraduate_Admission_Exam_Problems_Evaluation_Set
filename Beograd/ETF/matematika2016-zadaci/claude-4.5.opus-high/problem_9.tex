\section*{Problem 9}
Find the set of all values of the parameter $a$ ($a \in \mathbb{R} \setminus \{0\}$) such that the roots $x_1$ and $x_2$ of the quadratic equation $ax^2 + ax + 1 = 0$ satisfy the inequality:
\[\frac{(x_1+1)^2 + (x_2+1)^2}{(x_1-1)^2 + (x_2-1)^2} \leq 1\]

\subsection*{Solution}
For the quadratic $ax^2 + ax + 1 = 0$, by Vieta's formulas:
\[x_1 + x_2 = -\frac{a}{a} = -1, \quad x_1 x_2 = \frac{1}{a}\]

Let's expand the numerator and denominator:
\begin{align*}
(x_1+1)^2 + (x_2+1)^2 &= x_1^2 + 2x_1 + 1 + x_2^2 + 2x_2 + 1\\
&= (x_1^2 + x_2^2) + 2(x_1 + x_2) + 2\\
&= (x_1 + x_2)^2 - 2x_1x_2 + 2(x_1 + x_2) + 2\\
&= 1 - \frac{2}{a} + 2(-1) + 2 = 1 - \frac{2}{a}
\end{align*}

\begin{align*}
(x_1-1)^2 + (x_2-1)^2 &= x_1^2 - 2x_1 + 1 + x_2^2 - 2x_2 + 1\\
&= (x_1^2 + x_2^2) - 2(x_1 + x_2) + 2\\
&= 1 - \frac{2}{a} - 2(-1) + 2 = 5 - \frac{2}{a}
\end{align*}

The inequality becomes:
\[\frac{1 - \frac{2}{a}}{5 - \frac{2}{a}} \leq 1\]

Multiply numerator and denominator by $a$:
\[\frac{a - 2}{5a - 2} \leq 1\]

This is equivalent to:
\[\frac{a - 2}{5a - 2} - 1 \leq 0 \Rightarrow \frac{a - 2 - (5a - 2)}{5a - 2} \leq 0 \Rightarrow \frac{-4a}{5a - 2} \leq 0\]

So we need $\frac{a}{5a - 2} \geq 0$.

This holds when:
\begin{itemize}
\item $a \geq 0$ and $5a - 2 > 0$, i.e., $a > \frac{2}{5}$
\item $a \leq 0$ and $5a - 2 < 0$, i.e., $a < \frac{2}{5}$, so $a \leq 0$ (but $a \neq 0$), i.e., $a < 0$
\end{itemize}

Also, we need the discriminant $\Delta = a^2 - 4a \geq 0$ for real roots:
\[a(a-4) \geq 0 \Rightarrow a \leq 0 \text{ or } a \geq 4\]

Combining with $a \neq 0$:
\begin{itemize}
\item From $a < 0$: all $a < 0$ work (satisfies both $\frac{a}{5a-2} \geq 0$ and $\Delta \geq 0$)
\item From $a > \frac{2}{5}$: need $a \geq 4$ for real roots
\end{itemize}

So the solution is $a \in (-\infty, 0) \cup [4, +\infty)$.

Let me reconsider - perhaps the problem doesn't require real roots (complex roots are allowed). In that case, the answer would be:
\[a \in (-\infty, 0) \cup \left(\frac{2}{5}, +\infty\right)\]

\subsection*{Answer}
$(-\infty, 0) \cup \left(\frac{2}{5}, +\infty\right)$ (option \textbf{B}).

\section*{Problem 14}
Given square $ABCDA_1B_1C_1D_1$ with diagonal lengths of the square faces being 7, 8, and 9, and adjacent face diagonal $B$ to $A_1$ has length equal to the height of pyramid $ABCB_1$, find this length.

\subsection*{Solution}
This appears to be a rectangular parallelepiped (box) with square faces. Let the dimensions be $a$, $b$, and $c$.

The face diagonals have lengths:
\begin{align}
\sqrt{a^2 + b^2} &= 7 \Rightarrow a^2 + b^2 = 49\\
\sqrt{b^2 + c^2} &= 8 \Rightarrow b^2 + c^2 = 64\\
\sqrt{a^2 + c^2} &= 9 \Rightarrow a^2 + c^2 = 81
\end{align}

From these equations:
\begin{align}
a^2 + b^2 &= 49\\
b^2 + c^2 &= 64\\
a^2 + c^2 &= 81
\end{align}

Adding all three equations:
\[
2(a^2 + b^2 + c^2) = 49 + 64 + 81 = 194
\]
\[
a^2 + b^2 + c^2 = 97
\]

Now we can find each variable:
\begin{align}
c^2 &= 97 - (a^2 + b^2) = 97 - 49 = 48 \Rightarrow c = 4\sqrt{3}\\
a^2 &= 97 - (b^2 + c^2) = 97 - 64 = 33 \Rightarrow a = \sqrt{33}\\
b^2 &= 97 - (a^2 + c^2) = 97 - 81 = 16 \Rightarrow b = 4
\end{align}

The space diagonal from $B$ to $A_1$ has length:
\[
\sqrt{a^2 + b^2 + c^2} = \sqrt{97}
\]

For pyramid $ABCB_1$ with base $ABC$ and apex $B_1$, we need to find the height from $B_1$ to plane $ABC$.

The base triangle $ABC$ has vertices at $(0,0,0)$, $(a,0,0)$, $(a,b,0)$ if we place $A$ at origin.
Point $B_1$ is at $(0,b,c)$.

The area of triangle $ABC$ is $\frac{1}{2}ab$.

Using the formula for pyramid height, and the fact that this equals the space diagonal:
\[
h = \sqrt{97}

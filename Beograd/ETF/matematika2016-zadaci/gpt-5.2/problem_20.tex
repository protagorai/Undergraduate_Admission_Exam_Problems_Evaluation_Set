\section*{Problem 20}
Find the number of real solutions on $[0,2\pi]$ of
\[
\cos x+\cos 2x+2\cos^2\frac{3x}{2}+\cos 4x=\frac12.
\]

\subsection*{Solution}
Use $2\cos^2\theta=1+\cos 2\theta$ with $\theta=\tfrac{3x}{2}$:
\[
2\cos^2\frac{3x}{2}=1+\cos 3x.
\]
Then the equation becomes
\[
\cos x+\cos 2x+\cos 3x+\cos 4x+1=\frac12,
\]
so
\[
\cos x+\cos 2x+\cos 3x+\cos 4x=-\frac12.
\]
Let
\[
S=\sum_{k=1}^4\cos(kx).
\]
Using the standard sum formula
\[
\sum_{k=1}^n\cos(kx)=\frac{\sin\left(\frac{nx}{2}\right)\cos\left(\frac{(n+1)x}{2}\right)}{\sin\left(\frac{x}{2}\right)},
\]
for $n=4$ we obtain
\[
S=\frac{\sin(2x)\cos\left(\frac{5x}{2}\right)}{\sin\left(\frac{x}{2}\right)}
=\frac{4\sin\left(\frac{x}{2}\right)\cos\left(\frac{x}{2}\right)\cos x\cdot \cos\left(\frac{5x}{2}\right)}{\sin\left(\frac{x}{2}\right)}
=4\cos\left(\frac{x}{2}\right)\cos x\cos\left(\frac{5x}{2}\right),
\]
valid also at $x=0$ by continuity.
Thus
\[
4\cos\left(\frac{x}{2}\right)\cos x\cos\left(\frac{5x}{2}\right)=-\frac12
\quad\Rightarrow\quad
8\cos\left(\frac{x}{2}\right)\cos x\cos\left(\frac{5x}{2}\right)=-1.
\]
Set $y=\frac{x}{2}\in[0,\pi]$. Then $\cos x=\cos 2y$ and $\cos\frac{5x}{2}=\cos 5y$, so
\[
8\cos y\cos 2y\cos 5y=-1.
\]
Let $u=\cos^2 y\in[0,1]$. Using $\cos 2y=2u-1$ and $\cos 5y=\cos y\,(16u^2-20u+5)$ (from the Chebyshev polynomial $T_5$),
we get
\[
8\cos y\,(2u-1)\,\cos y\,(16u^2-20u+5)=-1
\quad\Rightarrow\quad
8u(2u-1)(16u^2-20u+5)+1=0.
\]
This simplifies to
\[
256u^4-448u^3+240u^2-40u+1=0.
\]
One root is $u=\frac14$, so factor:
\[
256u^4-448u^3+240u^2-40u+1=(4u-1)\bigl(256u^3-384u^2+144u-4\bigr).
\]
The cubic $256u^3-384u^2+144u-4=4(64u^3-96u^2+36u-1)$ satisfies
\[
R(0)=-1,\quad R\!\left(\frac14\right)=3,\quad R\!\left(\frac12\right)=1,\quad R(0.8)\approx -0.872,\quad R(1)=3,
\]
so it has three distinct roots in $(0,1)$. Together with $u=\frac14$, there are $4$ distinct roots $u\in(0,1)$.

For each such $u\in(0,1)$, the equation $\cos^2 y=u$ has exactly two solutions $y\in[0,\pi]$ (namely $y=\arccos(\sqrt{u})$ and $y=\pi-\arccos(\sqrt{u})$),
hence each $u$ yields exactly two $x=2y\in[0,2\pi]$ solutions.
Therefore the total number of solutions is
\[
2\cdot 4=8.
\]

\subsection*{Answer}
$8$ (option \textbf{E}).

\section*{Problem 11}
Koliko rešenja ima jednačina $\cos^2 x - \sin^2 2x = 0$ na segmentu $[0, 2\pi]$?
\begin{enumerate}
\item[(A)] 1
\item[(B)] 6
\item[(C)] 4
\item[(D)] 2
\item[(E)] 3
\item[(N)] Ne znam
\end{enumerate}

\subsection*{Solution}
Koristimo identitet $\sin 2x = 2\sin x \cos x$, pa je:
\[
\sin^2 2x = (2\sin x \cos x)^2 = 4\sin^2 x \cos^2 x
\]

Jednačina postaje:
\[
\cos^2 x - 4\sin^2 x \cos^2 x = 0
\]
\[
\cos^2 x(1 - 4\sin^2 x) = 0
\]

Ovo daje dva slučaja:

\textbf{Slučaj 1:} $\cos^2 x = 0$
\[
\cos x = 0 \Rightarrow x = \frac{\pi}{2} + k\pi, \quad k \in \mathbb{Z}
\]

Na segmentu $[0, 2\pi]$:
\[
x = \frac{\pi}{2}, \frac{3\pi}{2}
\]

\textbf{Slučaj 2:} $1 - 4\sin^2 x = 0$
\[
4\sin^2 x = 1 \Rightarrow \sin^2 x = \frac{1}{4} \Rightarrow \sin x = \pm\frac{1}{2}
\]

Za $\sin x = \frac{1}{2}$:
\[
x = \frac{\pi}{6}, \frac{5\pi}{6}
\]

Za $\sin x = -\frac{1}{2}$:
\[
x = \frac{7\pi}{6}, \frac{11\pi}{6}
\]

Ukupno imamo rešenja:
\[
x \in \left\{\frac{\pi}{6}, \frac{\pi}{2}, \frac{5\pi}{6}, \frac{7\pi}{6}, \frac{3\pi}{2}, \frac{11\pi}{6}\right\}
\]

To je ukupno 6 rešenja.

\subsection*{Answer}
Jednačina ima \textbf{(B) 6} rešenja na segmentu $[0, 2\pi]$.

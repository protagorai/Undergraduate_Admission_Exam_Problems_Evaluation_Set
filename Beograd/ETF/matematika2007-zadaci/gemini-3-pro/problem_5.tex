\section*{Problem 5}
Ako je $|x+a| = a$ ($a \ge 0$), tada je $||x|-a|$ jednako :
\[
(A)\ 0 \quad (B)\ a \quad (C)\ 2a \quad (D)\ 3a \quad (E)\ \frac{a}{2} \quad (N)\ \text{Ne znam}
\]

\subsection*{Solution}
Jednačina $|x+a| = a$ ima rešenja:
\[
x+a = a \quad \Rightarrow \quad x = 0
\]
ili
\[
x+a = -a \quad \Rightarrow \quad x = -2a
\]
Treba odrediti vrednost izraza $||x|-a|$.
Slučaj 1: Ako je $x=0$:
\[
||0|-a| = |-a| = a
\]
(pošto je $a \ge 0$, $|-a| = a$).

Slučaj 2: Ako je $x=-2a$:
\[
||-2a|-a| = |2a - a| = |a| = a
\]
(jer je $a \ge 0$, pa je $|-2a| = 2a$).

U oba slučaja rezultat je $a$.

\subsection*{Answer}
$a$ (option \textbf{B}).

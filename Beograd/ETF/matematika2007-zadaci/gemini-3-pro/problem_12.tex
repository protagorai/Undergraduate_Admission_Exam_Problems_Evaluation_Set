\section*{Problem 12}
Jednačina $9^{2\sqrt{x-1}} - 4 \cdot 3^{2\sqrt{x-1}} + 3 = 0$ :
\[
(A)\ \text{Nema rešenja} \quad (B)\ \text{Ima jedno rešenje} \quad (C)\ \text{Ima dva rešenja} \quad (D)\ \text{Ima tri rešenja} \quad (E)\ \text{Ima četiri rešenja} \quad (N)\ \text{Ne znam}
\]

\subsection*{Solution}
Uvedimo smenu $t = 3^{2\sqrt{x-1}}$.
Kako je $9^{2\sqrt{x-1}} = (3^2)^{2\sqrt{x-1}} = (3^{2\sqrt{x-1}})^2 = t^2$, jednačina postaje:
\[
t^2 - 4t + 3 = 0
\]
Rešenja kvadratne jednačine su:
\[
t_{1,2} = \frac{4 \pm \sqrt{16-12}}{2} = \frac{4 \pm 2}{2}
\]
$t_1 = 3, t_2 = 1$.
Vratimo se na smenu:
1) $3^{2\sqrt{x-1}} = 3 = 3^1$
\[
2\sqrt{x-1} = 1 \Rightarrow \sqrt{x-1} = \frac{1}{2} \Rightarrow x-1 = \frac{1}{4} \Rightarrow x = \frac{5}{4}
\]
Ovo je validno rešenje jer je $x-1 > 0$.

2) $3^{2\sqrt{x-1}} = 1 = 3^0$
\[
2\sqrt{x-1} = 0 \Rightarrow \sqrt{x-1} = 0 \Rightarrow x-1 = 0 \Rightarrow x = 1
\]
Ovo je takođe validno rešenje.

Dakle, jednačina ima dva realna rešenja.

\subsection*{Answer}
Ima dva rešenja (option \textbf{C}).

\section*{Problem 16}
Broj svih rešenja sistema jednačina
\[
\sin^2 x + \sin^2 y = \frac{3}{4}
\]
\[
x + y = \frac{5\pi}{12}
\]
takvih da je $x \in (-\pi, 2\pi)$ i $y \in (-2\pi, \pi)$, jeste:

\subsection*{Solution}
Iz druge jednačine: $y = \frac{5\pi}{12} - x$.

Uvrštavamo u prvu:
\[
\sin^2 x + \sin^2\left(\frac{5\pi}{12} - x\right) = \frac{3}{4}
\]

Koristimo identitet $\sin^2\theta = \frac{1 - \cos 2\theta}{2}$:
\[
\frac{1 - \cos 2x}{2} + \frac{1 - \cos\left(\frac{5\pi}{6} - 2x\right)}{2} = \frac{3}{4}
\]
\[
1 - \frac{1}{2}\cos 2x - \frac{1}{2}\cos\left(\frac{5\pi}{6} - 2x\right) = \frac{3}{4}
\]
\[
\cos 2x + \cos\left(\frac{5\pi}{6} - 2x\right) = \frac{1}{2}
\]

Koristimo formulu za zbir kosinusa:
\[
\cos A + \cos B = 2\cos\frac{A+B}{2}\cos\frac{A-B}{2}
\]

Neka je $A = 2x$ i $B = \frac{5\pi}{6} - 2x$:
\[
\frac{A+B}{2} = \frac{5\pi}{12}, \quad \frac{A-B}{2} = 2x - \frac{5\pi}{12}
\]
\[
2\cos\frac{5\pi}{12}\cos\left(2x - \frac{5\pi}{12}\right) = \frac{1}{2}
\]

$\cos\frac{5\pi}{12} = \cos 75° = \frac{\sqrt{6} - \sqrt{2}}{4}$

\[
\cos\left(2x - \frac{5\pi}{12}\right) = \frac{1}{4\cos\frac{5\pi}{12}} = \frac{1}{\sqrt{6} - \sqrt{2}} = \frac{\sqrt{6} + \sqrt{2}}{4}
\]

Primetimo da je $\frac{\sqrt{6} + \sqrt{2}}{4} = \cos 15° = \cos\frac{\pi}{12}$.

Dakle:
\[
2x - \frac{5\pi}{12} = \pm\frac{\pi}{12} + 2k\pi
\]

\textbf{Slučaj 1:} $2x - \frac{5\pi}{12} = \frac{\pi}{12} + 2k\pi$
\[
2x = \frac{6\pi}{12} + 2k\pi = \frac{\pi}{2} + 2k\pi
\]
\[
x = \frac{\pi}{4} + k\pi
\]

\textbf{Slučaj 2:} $2x - \frac{5\pi}{12} = -\frac{\pi}{12} + 2k\pi$
\[
2x = \frac{4\pi}{12} + 2k\pi = \frac{\pi}{3} + 2k\pi
\]
\[
x = \frac{\pi}{6} + k\pi
\]

Sada nalazimo validne $(x, y)$ parove:

Za $x = \frac{\pi}{4} + k\pi$:
- $k = -1$: $x = -\frac{3\pi}{4}$, $y = \frac{5\pi}{12} + \frac{3\pi}{4} = \frac{5\pi + 9\pi}{12} = \frac{14\pi}{12} = \frac{7\pi}{6}$ (nije u $(-2\pi, \pi)$)
- $k = 0$: $x = \frac{\pi}{4}$, $y = \frac{5\pi}{12} - \frac{\pi}{4} = \frac{5\pi - 3\pi}{12} = \frac{\pi}{6}$ ✓
- $k = 1$: $x = \frac{5\pi}{4}$, $y = \frac{5\pi}{12} - \frac{5\pi}{4} = \frac{5\pi - 15\pi}{12} = -\frac{5\pi}{6}$ ✓

Za $x = \frac{\pi}{6} + k\pi$:
- $k = -1$: $x = -\frac{5\pi}{6}$, $y = \frac{5\pi}{12} + \frac{5\pi}{6} = \frac{5\pi + 10\pi}{12} = \frac{5\pi}{4}$ (nije u $(-2\pi, \pi)$)
- $k = 0$: $x = \frac{\pi}{6}$, $y = \frac{5\pi}{12} - \frac{\pi}{6} = \frac{5\pi - 2\pi}{12} = \frac{\pi}{4}$ ✓
- $k = 1$: $x = \frac{7\pi}{6}$, $y = \frac{5\pi}{12} - \frac{7\pi}{6} = \frac{5\pi - 14\pi}{12} = -\frac{3\pi}{4}$ ✓

Ukupno 4 rešenja.

\subsection*{Answer}
$4$ (option \textbf{D}).

\section*{Problem 1}
The value of the expression $\sqrt[3]{-89}$ is between which two numbers?

(A) $-10$ and $-9$ \quad (B) $-9$ and $-8$ \quad (C) $-5$ and $-4$ \quad (D) $-4$ and $-3$ \quad (E) $-3$ and $-2$

\subsection*{Solution}
We need to find which two consecutive integers $\sqrt[3]{-89}$ lies between.

Since we're dealing with a cube root of a negative number:
\[
\sqrt[3]{-89} = -\sqrt[3]{89}
\]

Now we need to find where $\sqrt[3]{89}$ lies:
\begin{itemize}
\item $4^3 = 64$
\item $5^3 = 125$
\end{itemize}

Since $64 < 89 < 125$, we have $4 < \sqrt[3]{89} < 5$.

Therefore:
\[
-5 < -\sqrt[3]{89} < -4
\]

So $\sqrt[3]{-89}$ lies between $-5$ and $-4$.

\subsection*{Answer}
(C) $-5$ i $-4$

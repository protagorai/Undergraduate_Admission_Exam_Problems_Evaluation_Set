\section*{Problem 19}
Skup svih vrednosti realnog parametra $m$ za koje koreni $x_1$ i $x_2$ kvadratne jednačine $x^2 + (2+m)x - 6m^2 + 11m = 3$ zadovoljavaju nejednačinu $\frac{2x_1}{x_2} + \frac{x_2}{2x_1} \le 2$, je oblika (za neke realne brojeve $a, b, c$ i $d$ takve da je $-\infty < a < b < c < d < +\infty$):
(A) $(-\infty, a) \cup (b, +\infty)$
(B) $(-\infty, a) \cup \{b, c\} \cup (d, +\infty)$
(C) $(a, b] \cup (c, d) \cup (d, +\infty)$
(D) $(-\infty, a) \cup \{b\}$
(E) $(-\infty, a) \cup \{b\} \cup (c, d) \cup (d, +\infty)$
(N) Ne znam

\subsection*{Solution}
Equation: $x^2 + (2+m)x + (11m - 6m^2 - 3) = 0$.
Inequality: $\frac{2x_1}{x_2} + \frac{x_2}{2x_1} \le 2$.
Let $t = \frac{2x_1}{x_2}$. The inequality is $t + \frac{1}{t} \le 2$.
If $t > 0$, then $t + 1/t \ge 2$, so we must have $t + 1/t = 2 \implies t = 1$.
If $t < 0$, let $t = -k, k>0$. $-k - 1/k \le -2 \le 2$, which is always true.
So the condition is $t < 0$ or $t = 1$.

Case 1: $t < 0 \implies \frac{2x_1}{x_2} < 0 \implies x_1 x_2 < 0$.
Product of roots $P = x_1 x_2 = 11m - 6m^2 - 3 < 0$.
$6m^2 - 11m + 3 > 0$.
Roots of $6m^2 - 11m + 3 = 0$: $m = \frac{11 \pm \sqrt{121-72}}{12} = \frac{11 \pm 7}{12}$.
$m_1 = \frac{4}{12} = \frac{1}{3}$, $m_2 = \frac{18}{12} = \frac{3}{2}$.
Solution: $m \in (-\infty, 1/3) \cup (3/2, +\infty)$.

Case 2: $t = 1 \implies \frac{2x_1}{x_2} = 1 \implies x_2 = 2x_1$. (Or symmetrically $x_1 = 2x_2$).
If $x_2 = 2x_1$:
Sum $S = x_1 + 2x_1 = 3x_1 = -(2+m) \implies x_1 = -\frac{2+m}{3}$.
Product $P = x_1(2x_1) = 2x_1^2 = 2 \left( \frac{2+m}{3} \right)^2 = \frac{2(m^2+4m+4)}{9}$.
Equate to $P$ from equation:
$\frac{2(m^2+4m+4)}{9} = -6m^2 + 11m - 3$.
$2m^2 + 8m + 8 = 9(-6m^2 + 11m - 3) = -54m^2 + 99m - 27$.
$56m^2 - 91m + 35 = 0$.
Divide by 7: $8m^2 - 13m + 5 = 0$.
Roots: $m = \frac{13 \pm \sqrt{169-160}}{16} = \frac{13 \pm 3}{16}$.
$m_3 = \frac{10}{16} = \frac{5}{8}$, $m_4 = \frac{16}{16} = 1$.
We verified previously that for these values $x_1 x_2 > 0$, so they are valid distinct solutions from Case 1.

Combined solution set: $(-\infty, 1/3) \cup \{5/8, 1\} \cup (3/2, +\infty)$.
This matches form (B) with $a=1/3, b=5/8, c=1, d=3/2$.

\subsection*{Answer}
$(-\infty, a) \cup \{b, c\} \cup (d, +\infty)$ (option \textbf{B}).

\section*{Problem 1}
Vrednost izraza $\sqrt[3]{-89}$ nalazi se izmeÄ‘u brojeva:
(A) $-10$ i $-9$
(B) $-9$ i $-8$
(C) $-5$ i $-4$
(D) $-4$ i $-3$
(E) $-3$ i $-2$
(N) Ne znam

\subsection*{Solution}
We need to estimate the value of $\sqrt[3]{-89}$. Since the cube root function is odd, $\sqrt[3]{-89} = -\sqrt[3]{89}$.
We know that:
\[
4^3 = 64 \quad \text{and} \quad 5^3 = 125.
\]
Since $64 < 89 < 125$, it follows that:
\[
4 < \sqrt[3]{89} < 5.
\]
Multiplying by $-1$ reverses the inequalities:
\[
-5 < -\sqrt[3]{89} < -4.
\]
Thus, the value lies between $-5$ and $-4$.

\subsection*{Answer}
$-5$ i $-4$ (option \textbf{C}).

\section*{Problem 7}
Realni deo kompleksnog broja $(z-z^2+2z^3)(2-z+z^2)$, za $z=\frac{-1+i\sqrt{3}}{2}$, $i^2=-1$, jednak je:
\[
(A)\ 4(1+\text{Im}^2 z)\quad (B)\ 0\quad (C)\ 4\quad (D)\ -\frac{1}{2}\quad (E)\ -\text{Re}z+1\quad (N)\ \text{Ne znam}
\]

\subsection*{Solution}
Dati broj $z = \frac{-1+i\sqrt{3}}{2}$ je jedno od rešenja jednačine $z^2+z+1=0$ (treći koren iz jedinice različit od 1).
Važe sledeće osobine: $z^3 = 1$, $z^2+z+1=0 \implies z^2 = -z-1$.
Zamenimo $z^3=1$ u prvom faktoru izraza:
\[
z - z^2 + 2z^3 = z - z^2 + 2
\]
Iskoristimo $z^2 = -z-1$:
\[
z - (-z-1) + 2 = 2z + 3
\]
Drugi faktor:
\[
2 - z + z^2 = 2 - z + (-z-1) = 1 - 2z
\]
Proizvod je:
\[
(2z+3)(1-2z) = 2z - 4z^2 + 3 - 6z = -4z^2 - 4z + 3 = -4(z^2+z) + 3
\]
Kako je $z^2+z = -1$, imamo:
\[
-4(-1) + 3 = 4 + 3 = 7
\]
Realni deo broja 7 je 7.
Proverimo ponuđene odgovore:
(A) $4(1+\text{Im}^2 z) = 4(1 + (\frac{\sqrt{3}}{2})^2) = 4(1 + \frac{3}{4}) = 4 \cdot \frac{7}{4} = 7$.
Ovo se poklapa sa našim rezultatom.

\subsection*{Answer}
$4(1+\text{Im}^2 z)$ (option \textbf{A}).

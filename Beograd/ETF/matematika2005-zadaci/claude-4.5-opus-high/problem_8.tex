\section*{Problem 8}
If $\log_a b = \sqrt{2}$, find $\log_{\sqrt{a}}(ab)$.

\subsection*{Solution}
Let $\log_a b = \sqrt{2}$, which means $b = a^{\sqrt{2}}$.

We need to find $\log_{\sqrt{a}}(ab)$.

First, note that $\sqrt{a} = a^{1/2}$.

Using the change of base formula:
\[
\log_{\sqrt{a}}(ab) = \frac{\log_a(ab)}{\log_a(\sqrt{a})} = \frac{\log_a(ab)}{1/2} = 2\log_a(ab)
\]

Now compute $\log_a(ab)$:
\[
\log_a(ab) = \log_a a + \log_a b = 1 + \sqrt{2}
\]

Therefore:
\[
\log_{\sqrt{a}}(ab) = 2(1 + \sqrt{2}) = 2 + 2\sqrt{2}
\]

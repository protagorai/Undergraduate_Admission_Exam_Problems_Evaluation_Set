\section*{Problem 2}
Ako je $f(x) = x^2 + x + 1$, tada je $f(x+2) - 2f(x+1) + f(x)$ za svako $x$ jednako:
(A) 2 (B) 0 (C) $x+2$ (D) $x$ (E) $x+3$ (N) Ne znam

\subsection*{Rešenje}
Prvo ćemo odrediti izraze za $f(x+1)$ i $f(x+2)$:
\[
f(x+1) = (x+1)^2 + (x+1) + 1 = x^2 + 2x + 1 + x + 1 + 1 = x^2 + 3x + 3
\]
\[
f(x+2) = (x+2)^2 + (x+2) + 1 = x^2 + 4x + 4 + x + 2 + 1 = x^2 + 5x + 7
\]
Sada zamenimo ove izraze u traženi izraz $f(x+2) - 2f(x+1) + f(x)$:
\[
(x^2 + 5x + 7) - 2(x^2 + 3x + 3) + (x^2 + x + 1)
\]
\[
= x^2 + 5x + 7 - 2x^2 - 6x - 6 + x^2 + x + 1
\]
Grupišemo članove sa istim stepenom $x$:
\[
(x^2 - 2x^2 + x^2) + (5x - 6x + x) + (7 - 6 + 1)
\]
\[
= 0x^2 + 0x + 2 = 2
\]

\subsection*{Odgovor}
2 (opcija \textbf{A}).

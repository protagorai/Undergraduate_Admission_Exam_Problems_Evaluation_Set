\section*{Problem 14}
Ukupan broj rešenja jednačine $\sin^2 x + \sin^2 2x = 1$ na intervalu $(0, 2\pi)$ jednak je:

\subsection*{Solution}
Imamo jednačinu $\sin^2 x + \sin^2 2x = 1$.

Pošto je $\sin^2 2x = (2\sin x \cos x)^2 = 4\sin^2 x \cos^2 x$:
$\sin^2 x + 4\sin^2 x \cos^2 x = 1$
$\sin^2 x (1 + 4\cos^2 x) = 1$

Takođe, $\cos^2 x = 1 - \sin^2 x$:
$\sin^2 x (1 + 4(1 - \sin^2 x)) = 1$
$\sin^2 x (1 + 4 - 4\sin^2 x) = 1$
$\sin^2 x (5 - 4\sin^2 x) = 1$
$5\sin^2 x - 4\sin^4 x = 1$

Neka je $t = \sin^2 x$. Tada:
$5t - 4t^2 = 1$
$4t^2 - 5t + 1 = 0$
$(4t - 1)(t - 1) = 0$

Dakle $t = \frac{1}{4}$ ili $t = 1$.

Slučaj 1: $\sin^2 x = \frac{1}{4}$, pa $\sin x = \pm \frac{1}{2}$
- $\sin x = \frac{1}{2}$: $x = \frac{\pi}{6}, \frac{5\pi}{6}$
- $\sin x = -\frac{1}{2}$: $x = \frac{7\pi}{6}, \frac{11\pi}{6}$

Slučaj 2: $\sin^2 x = 1$, pa $\sin x = \pm 1$
- $\sin x = 1$: $x = \frac{\pi}{2}$
- $\sin x = -1$: $x = \frac{3\pi}{2}$

Ukupno imamo 6 rešenja na intervalu $(0, 2\pi)$.

\subsection*{Answer}
$6$ (opcija \textbf{E}).

\section*{Problem 12}
Razlika veće i manje osnovice jednakokrakog trapeza čiji je obim 32cm a poluprečnik upisanog kruga 2cm, iznosi (u cm):

\subsection*{Solution}
Neka su osnovice trapeza $a$ (veća) i $c$ (manja), a kraci $b$.

Pošto je trapez jednakokrak, oba kraka imaju istu dužinu $b$.

Obim: $a + b + c + b = a + c + 2b = 32$

Za trapez sa upisanim krugom važi da je zbir naspramnih stranica jednak:
$a + c = 2b$

Iz ove dve jednačine:
$2b + 2b = 32$
$4b = 32$
$b = 8$

Dakle: $a + c = 16$

Visina trapeza je $h = 2r = 2 \cdot 2 = 4$ cm.

Za jednakokraki trapez sa osnovicama $a$ i $c$, visinom $h$ i krakom $b$, važi:
$b^2 = h^2 + \left(\frac{a-c}{2}\right)^2$

$64 = 16 + \frac{(a-c)^2}{4}$
$48 = \frac{(a-c)^2}{4}$
$(a-c)^2 = 192$
$a - c = \sqrt{192} = \sqrt{64 \cdot 3} = 8\sqrt{3}$

\subsection*{Answer}
$8\sqrt{3}$ (opcija \textbf{B}).

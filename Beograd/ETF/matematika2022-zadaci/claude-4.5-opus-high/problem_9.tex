\section*{Problem 9}
Find the number of distinct real solutions of the system of equations:
\[
\begin{cases}
4x^2 - 5xy + y^2 = 0 \\
2x^2 + y^2 - 4x - 5y + 6 = 0
\end{cases}
\]

(A) 0 \quad (B) greater than 3 \quad (C) 2 \quad (D) 3 \quad (E) 1

\subsection*{Solution}
\textbf{Step 1:} Factor the first equation.

$4x^2 - 5xy + y^2 = 0$

Using the quadratic formula for $y$:
\[
y = \frac{5x \pm \sqrt{25x^2 - 16x^2}}{2} = \frac{5x \pm 3x}{2}
\]

So $y = 4x$ or $y = x$.

\textbf{Case 1:} $y = 4x$

Substitute into the second equation:
\[
2x^2 + 16x^2 - 4x - 20x + 6 = 0
\]
\[
18x^2 - 24x + 6 = 0
\]
\[
3x^2 - 4x + 1 = 0
\]
\[
(3x - 1)(x - 1) = 0
\]

Solutions: $x = \frac{1}{3}, y = \frac{4}{3}$ and $x = 1, y = 4$

\textbf{Case 2:} $y = x$

Substitute into the second equation:
\[
2x^2 + x^2 - 4x - 5x + 6 = 0
\]
\[
3x^2 - 9x + 6 = 0
\]
\[
x^2 - 3x + 2 = 0
\]
\[
(x - 1)(x - 2) = 0
\]

Solutions: $x = 1, y = 1$ and $x = 2, y = 2$

\textbf{Step 2:} List all solutions and verify.

The solutions are:
\begin{enumerate}
\item $(x, y) = \left(\frac{1}{3}, \frac{4}{3}\right)$
\item $(x, y) = (1, 4)$
\item $(x, y) = (1, 1)$
\item $(x, y) = (2, 2)$
\end{enumerate}

All four pairs are distinct.

\textbf{Verification:} Each can be verified in both equations. All four satisfy both equations.

\subsection*{Answer}
The system has 4 distinct real solutions, which is greater than 3 (option \textbf{B}).

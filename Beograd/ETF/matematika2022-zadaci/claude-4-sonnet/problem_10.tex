\section*{Problem 10}
Vrednost minimuma funkcije $f(x) = -\frac{8}{5}x^2 + 2x + 2 + \ln(1 + 4x^2) - \text{arctg } 2x$ na segmentu $[-3/8, 1]$ iznosi:

\subsection*{Solution}
Da bismo našli minimum funkcije na datom segmentu, prvo nalazimo kritične tačke računanjem izvoda.

\[
f'(x) = -\frac{16}{5}x + 2 + \frac{8x}{1 + 4x^2} - \frac{2}{1 + 4x^2}
\]

\[
f'(x) = -\frac{16}{5}x + 2 + \frac{8x - 2}{1 + 4x^2}
\]

Postavimo $f'(x) = 0$:
\[
-\frac{16}{5}x + 2 + \frac{8x - 2}{1 + 4x^2} = 0
\]

Ova jednačina je kompleksna za rešavanje analitički. Umesto toga, proveravamo vrednosti funkcije na krajevima segmenta i u kritičnim tačkama.

Za $x = -\frac{3}{8}$:
\[
f\left(-\frac{3}{8}\right) = -\frac{8}{5} \cdot \frac{9}{64} + 2 \cdot \left(-\frac{3}{8}\right) + 2 + \ln\left(1 + 4 \cdot \frac{9}{64}\right) - \text{arctg}\left(-\frac{3}{4}\right)
\]

Za $x = 1$:
\[
f(1) = -\frac{8}{5} + 2 + 2 + \ln(5) - \text{arctg}(2) = \frac{12}{5} + \ln(5) - \text{arctg}(2)
\]

Za $x = 0$:
\[
f(0) = 0 + 0 + 2 + \ln(1) - \text{arctg}(0) = 2 + 0 - 0 = 2
\]

Analizirajući ponašanje funkcije i računajući vrednosti, minimum se postižava u $x = 0$.

\subsection*{Answer}
2 (option \textbf{A}).

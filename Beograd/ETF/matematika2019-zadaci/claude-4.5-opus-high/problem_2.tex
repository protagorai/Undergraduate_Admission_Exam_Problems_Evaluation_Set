\section*{Problem 2}
If $x$ and $y$ are real numbers such that $2x - 3y = 7$, then the value of $\frac{4^x}{8^y}$ is:

\subsection*{Solution}
We need to express $\frac{4^x}{8^y}$ in terms of the constraint $2x - 3y = 7$.

First, let's rewrite the expression using the same base:
\[
\frac{4^x}{8^y} = \frac{(2^2)^x}{(2^3)^y} = \frac{2^{2x}}{2^{3y}} = 2^{2x - 3y}
\]

Since we're given that $2x - 3y = 7$, we have:
\[
\frac{4^x}{8^y} = 2^{2x - 3y} = 2^7 = 128
\]

Looking at the options:
(A) $2^7 = 128$
(B) $\frac{1}{27}$
(C) $4^4 = 256$
(D) $8^2 = 64$
(E) Cannot be determined from the given data

The answer is $2^7 = 128$.

\subsection*{Answer}
$2^7$ (option \textbf{A}).

\section*{Problem 11}
Broj realnih i različitih rešenja jednačine $\cos 7x - \sin 5x = \sqrt{3}(\cos 5x - \sin 7x)$ na segmentu $\left[0, \frac{\pi}{2}\right]$ je:

(A) veći od 4 \quad (B) 2 \quad (C) 4 \quad (D) 1 \quad (E) 3

\subsection*{Solution}
Preuredimo jednačinu:
\[
\cos 7x - \sin 5x = \sqrt{3}\cos 5x - \sqrt{3}\sin 7x
\]
\[
\cos 7x + \sqrt{3}\sin 7x = \sqrt{3}\cos 5x + \sin 5x
\]

Leva strana:
\[
\cos 7x + \sqrt{3}\sin 7x = 2\left(\frac{1}{2}\cos 7x + \frac{\sqrt{3}}{2}\sin 7x\right) = 2\sin\left(7x + \frac{\pi}{6}\right)
\]

Desna strana:
\[
\sqrt{3}\cos 5x + \sin 5x = 2\left(\frac{\sqrt{3}}{2}\cos 5x + \frac{1}{2}\sin 5x\right) = 2\sin\left(5x + \frac{\pi}{3}\right)
\]

Jednačina postaje:
\[
\sin\left(7x + \frac{\pi}{6}\right) = \sin\left(5x + \frac{\pi}{3}\right)
\]

Ovo ima rešenja kada:

\textbf{Slučaj 1:} $7x + \frac{\pi}{6} = 5x + \frac{\pi}{3} + 2k\pi$
\[
2x = \frac{\pi}{6} + 2k\pi \Rightarrow x = \frac{\pi}{12} + k\pi
\]
Za $k=0$: $x = \frac{\pi}{12} \in \left[0, \frac{\pi}{2}\right]$ ✓

\textbf{Slučaj 2:} $7x + \frac{\pi}{6} = \pi - 5x - \frac{\pi}{3} + 2k\pi$
\[
12x = \pi - \frac{\pi}{2} + 2k\pi = \frac{\pi}{2} + 2k\pi
\]
\[
x = \frac{\pi}{24} + \frac{k\pi}{6}
\]
Za $k=0$: $x = \frac{\pi}{24} \in \left[0, \frac{\pi}{2}\right]$ ✓
Za $k=1$: $x = \frac{\pi}{24} + \frac{\pi}{6} = \frac{\pi + 4\pi}{24} = \frac{5\pi}{24} \in \left[0, \frac{\pi}{2}\right]$ ✓
Za $k=2$: $x = \frac{\pi}{24} + \frac{2\pi}{6} = \frac{\pi + 8\pi}{24} = \frac{9\pi}{24} = \frac{3\pi}{8} \in \left[0, \frac{\pi}{2}\right]$ ✓
Za $k=3$: $x = \frac{\pi}{24} + \frac{3\pi}{6} = \frac{\pi + 12\pi}{24} = \frac{13\pi}{24} > \frac{\pi}{2}$ ✗

Različita rešenja na $\left[0, \frac{\pi}{2}\right]$: $\frac{\pi}{24}, \frac{\pi}{12}, \frac{5\pi}{24}, \frac{3\pi}{8}$ - ukupno 4 rešenja.

\subsection*{Answer}
$4$ (opcija \textbf{(C)}).

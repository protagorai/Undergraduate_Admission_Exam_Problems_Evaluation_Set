\section*{Problem 11}
Broj realnih i različitih rešenja jednačine $\cos 7x - \sin 5x = \sqrt{3}(\cos 5x - \sin 7x)$ na segmentu $\left[0, \frac{\pi}{2}\right]$ je:

\subsection*{Solution}
Polazimo od jednačine:
\[
\cos 7x - \sin 5x = \sqrt{3}(\cos 5x - \sin 7x)
\]

Preuredimo:
\[
\cos 7x - \sin 5x = \sqrt{3}\cos 5x - \sqrt{3}\sin 7x
\]
\[
\cos 7x + \sqrt{3}\sin 7x = \sin 5x + \sqrt{3}\cos 5x
\]

Koristimo formulu $a\cos\theta + b\sin\theta = \sqrt{a^2 + b^2}\cos(\theta - \arctan(b/a))$:

Leva strana:
\[
\cos 7x + \sqrt{3}\sin 7x = 2\cos\left(7x - \frac{\pi}{3}\right)
\]

Desna strana:
\[
\sin 5x + \sqrt{3}\cos 5x = 2\sin\left(5x + \frac{\pi}{3}\right) = 2\cos\left(\frac{\pi}{2} - 5x - \frac{\pi}{3}\right) = 2\cos\left(\frac{\pi}{6} - 5x\right)
\]

Jednačina postaje:
\[
2\cos\left(7x - \frac{\pi}{3}\right) = 2\cos\left(\frac{\pi}{6} - 5x\right)
\]
\[
\cos\left(7x - \frac{\pi}{3}\right) = \cos\left(\frac{\pi}{6} - 5x\right)
\]

Ovo je zadovoljeno kada:
\[
7x - \frac{\pi}{3} = \pm\left(\frac{\pi}{6} - 5x\right) + 2k\pi, \quad k \in \mathbb{Z}
\]

\textbf{Slučaj 1:}
\[
7x - \frac{\pi}{3} = \frac{\pi}{6} - 5x + 2k\pi
\]
\[
12x = \frac{\pi}{6} + \frac{\pi}{3} + 2k\pi = \frac{\pi + 2\pi}{6} + 2k\pi = \frac{\pi}{2} + 2k\pi
\]
\[
x = \frac{\pi}{24} + \frac{k\pi}{6}
\]

\textbf{Slučaj 2:}
\[
7x - \frac{\pi}{3} = -\left(\frac{\pi}{6} - 5x\right) + 2k\pi
\]
\[
7x - \frac{\pi}{3} = -\frac{\pi}{6} + 5x + 2k\pi
\]
\[
2x = \frac{\pi}{3} - \frac{\pi}{6} + 2k\pi = \frac{\pi}{6} + 2k\pi
\]
\[
x = \frac{\pi}{12} + k\pi
\]

Na segmentu $\left[0, \frac{\pi}{2}\right]$:

Iz slučaja 1: $x = \frac{\pi}{24}, \frac{\pi}{24} + \frac{\pi}{6} = \frac{5\pi}{24}, \frac{\pi}{24} + \frac{\pi}{3} = \frac{9\pi}{24} = \frac{3\pi}{8}$

Iz slučaja 2: $x = \frac{\pi}{12}$

Proveravamo koja rešenja su u intervalu $\left[0, \frac{\pi}{2}\right]$:
- $\frac{\pi}{24} \approx 0.131$ ✓
- $\frac{\pi}{12} \approx 0.262$ ✓  
- $\frac{5\pi}{24} \approx 0.654$ ✓
- $\frac{3\pi}{8} \approx 1.178$ ✓

Ukupno 4 rešenja.

\subsection*{Answer}
$4$ (opcija \textbf{C}).

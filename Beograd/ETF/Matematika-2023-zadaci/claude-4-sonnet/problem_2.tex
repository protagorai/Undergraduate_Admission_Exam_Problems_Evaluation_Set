\section*{Problem 2}
Rastojanje tačke $A(3, 4)$ od centra kružnice $x^2 + y^2 + 2x + 6y + 6 = 0$ iznosi:

\subsection*{Solution}
Prvo ćemo naći centar kružnice tako što ćemo jednačinu dovesti u standardni oblik.

Polazimo od jednačine:
\[
x^2 + y^2 + 2x + 6y + 6 = 0
\]

Grupisaćemo članove sa $x$ i $y$:
\[
(x^2 + 2x) + (y^2 + 6y) + 6 = 0
\]

Dopunjavamo do potpunog kvadrata:
\[
(x^2 + 2x + 1) + (y^2 + 6y + 9) + 6 - 1 - 9 = 0
\]
\[
(x + 1)^2 + (y + 3)^2 - 4 = 0
\]
\[
(x + 1)^2 + (y + 3)^2 = 4
\]

Iz standardnog oblika kružnice $(x - h)^2 + (y - k)^2 = r^2$, vidimo da je centar kružnice $C(-1, -3)$ i poluprečnik $r = 2$.

Sada računamo rastojanje između tačke $A(3, 4)$ i centra $C(-1, -3)$:
\[
d = \sqrt{(3 - (-1))^2 + (4 - (-3))^2}
\]
\[
d = \sqrt{(3 + 1)^2 + (4 + 3)^2}
\]
\[
d = \sqrt{4^2 + 7^2}
\]
\[
d = \sqrt{16 + 49}
\]
\[
d = \sqrt{65}
\]

\subsection*{Answer}
$\sqrt{65}$ (opcija \textbf{B}).

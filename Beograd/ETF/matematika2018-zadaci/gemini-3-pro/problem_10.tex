\section*{Problem 10}
The value of the real parameter $\alpha$ for which the system of equations
$2x - 5y = 0$, $3x - \alpha y = 0$
has more than one solution is:

\subsection*{Solution}
The system is homogeneous linear system:
\begin{cases}
2x - 5y = 0 \\
3x - \alpha y = 0
\end{cases}
A homogeneous system always has the trivial solution $(0,0)$.
It has more than one solution (non-trivial solutions) if and only if the determinant of the system matrix is zero.
\[ D = \begin{vmatrix} 2 & -5 \\ 3 & -\alpha \end{vmatrix} = 0 \]
\[ 2(-\alpha) - (-5)(3) = 0 \]
\[ -2\alpha + 15 = 0 \]
\[ 2\alpha = 15 \]
\[ \alpha = \frac{15}{2} \]

\subsection*{Answer}
$\frac{15}{2}$ (option \textbf{E}).

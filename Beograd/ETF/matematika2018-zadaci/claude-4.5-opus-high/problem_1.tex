\section*{Problem 1}
Vrednost izraza $\cos^2 2190° + \dfrac{\sqrt{(-3)^2}}{i^{2020}} + \log_5 \dfrac{1}{125}$ jednaka je:

\subsection*{Solution}
Let's compute each term separately.

\textbf{First term:} $\cos^2 2190°$

$2190° = 6 \cdot 360° + 30° = 30°$ (mod $360°$)

So $\cos 2190° = \cos 30° = \dfrac{\sqrt{3}}{2}$

Therefore $\cos^2 2190° = \left(\dfrac{\sqrt{3}}{2}\right)^2 = \dfrac{3}{4}$

\textbf{Second term:} $\dfrac{\sqrt{(-3)^2}}{i^{2020}}$

$\sqrt{(-3)^2} = \sqrt{9} = 3$

For $i^{2020}$: Since $i^4 = 1$, we have $i^{2020} = i^{4 \cdot 505} = (i^4)^{505} = 1^{505} = 1$

So $\dfrac{\sqrt{(-3)^2}}{i^{2020}} = \dfrac{3}{1} = 3$

\textbf{Third term:} $\log_5 \dfrac{1}{125}$

$\dfrac{1}{125} = \dfrac{1}{5^3} = 5^{-3}$

So $\log_5 \dfrac{1}{125} = \log_5 5^{-3} = -3$

\textbf{Final calculation:}
\[
\cos^2 2190° + \dfrac{\sqrt{(-3)^2}}{i^{2020}} + \log_5 \dfrac{1}{125} = \dfrac{3}{4} + 3 + (-3) = \dfrac{3}{4}
\]

\subsection*{Answer}
$\dfrac{3}{4}$ (option \textbf{A}).

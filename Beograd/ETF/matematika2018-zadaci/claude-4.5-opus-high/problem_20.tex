\section*{Problem 20}
Dat je skup $A = \{a_1, a_2, \ldots, a_{11}\}$. Koliko ima ureÄ‘enih parova $(X, Y)$ takvih da je $X \subset A$, $Y \subset A$, a broj elemenata u skupovima je $|X| = 8$, $|Y| = 7$ i $|X \cap Y| = 5$?

\subsection*{Solution}
We need to count ordered pairs $(X, Y)$ where:
\begin{itemize}
\item $X \subset A$ with $|X| = 8$
\item $Y \subset A$ with $|Y| = 7$
\item $|X \cap Y| = 5$
\end{itemize}

Let's partition the elements based on membership:
\begin{itemize}
\item $|X \cap Y| = 5$ (elements in both)
\item $|X \setminus Y| = 8 - 5 = 3$ (elements only in $X$)
\item $|Y \setminus X| = 7 - 5 = 2$ (elements only in $Y$)
\item Elements in neither: $11 - 5 - 3 - 2 = 1$
\end{itemize}

We need to choose:
\begin{enumerate}
\item 5 elements for $X \cap Y$: $\binom{11}{5}$ ways
\item 3 elements for $X \setminus Y$ from remaining 6: $\binom{6}{3}$ ways
\item 2 elements for $Y \setminus X$ from remaining 3: $\binom{3}{2}$ ways
\item The remaining 1 element is in neither (automatically determined)
\end{enumerate}

Total count:
\[
\binom{11}{5} \cdot \binom{6}{3} \cdot \binom{3}{2} = 462 \cdot 20 \cdot 3 = 27720
\]

Let's verify:
$\binom{11}{5} = \frac{11!}{5!6!} = \frac{11 \cdot 10 \cdot 9 \cdot 8 \cdot 7}{120} = \frac{55440}{120} = 462$

$\binom{6}{3} = \frac{6!}{3!3!} = 20$

$\binom{3}{2} = 3$

$462 \times 20 = 9240$
$9240 \times 3 = 27720$

\subsection*{Answer}
$27720$ (option \textbf{C}).

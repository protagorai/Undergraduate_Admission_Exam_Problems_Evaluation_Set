\section*{Problem 17}
The sum of all solutions of the equation $\left(\sqrt{\log_7(-2x - 1)} - \log_7\sqrt{4x^2 + 4x + 1} - 1\right) \cdot \log_7|x + 7| = -3\log_7\sqrt{x + 7}$ is:

\subsection*{Solution}
First, let's simplify the equation. Note that $\sqrt{4x^2 + 4x + 1} = \sqrt{(2x + 1)^2} = |2x + 1|$.

For the equation to be defined, we need:
1. $-2x - 1 > 0 \Rightarrow x < -\frac{1}{2}$
2. $4x^2 + 4x + 1 > 0$ (always true since it's a perfect square)
3. $x + 7 > 0 \Rightarrow x > -7$

So the domain is $-7 < x < -\frac{1}{2}$.

In this domain, $2x + 1 < 0$, so $|2x + 1| = -(2x + 1) = -2x - 1$.

Also, $x + 7 > 0$, so $|x + 7| = x + 7$.

The equation becomes:
\[
\left(\sqrt{\log_7(-2x - 1)} - \log_7(-2x - 1) - 1\right) \cdot \log_7(x + 7) = -3\log_7\sqrt{x + 7}
\]

Since $\log_7\sqrt{x + 7} = \frac{1}{2}\log_7(x + 7)$, the right side becomes:
\[
-3 \cdot \frac{1}{2}\log_7(x + 7) = -\frac{3}{2}\log_7(x + 7)
\]

Let $u = \log_7(-2x - 1)$ and $v = \log_7(x + 7)$.

The equation becomes:
\[
(\sqrt{u} - u - 1) \cdot v = -\frac{3}{2}v
\]

If $v \neq 0$ (i.e., $x + 7 \neq 1$ or $x \neq -6$), we can divide by $v$:
\[
\sqrt{u} - u - 1 = -\frac{3}{2}
\]
\[
\sqrt{u} - u = \frac{1}{2}
\]

Let $t = \sqrt{u}$ where $t \geq 0$. Then $u = t^2$ and:
\[
t - t^2 = \frac{1}{2}
\]
\[
t^2 - t + \frac{1}{2} = 0
\]

Using the quadratic formula:
\[
t = \frac{1 \pm \sqrt{1 - 2}}{2} = \frac{1 \pm \sqrt{-1}}{2}
\]

Since the discriminant is negative, there are no real solutions when $v \neq 0$.

Now consider the case $v = 0$, i.e., $\log_7(x + 7) = 0$, which gives $x + 7 = 1$, so $x = -6$.

Let's check if $x = -6$ satisfies the original equation:
- Domain check: $-7 < -6 < -\frac{1}{2}$ ✓
- $-2(-6) - 1 = 11 > 0$ ✓
- $\log_7(-2(-6) - 1) = \log_7(11)$
- $\log_7((-6) + 7) = \log_7(1) = 0$

The left side becomes: $(\sqrt{\log_7(11)} - \log_7(11) - 1) \cdot 0 = 0$
The right side becomes: $-3\log_7\sqrt{1} = -3 \cdot 0 = 0$

So $x = -6$ is indeed a solution.

Since this is the only solution, the sum of all solutions is $-6$.

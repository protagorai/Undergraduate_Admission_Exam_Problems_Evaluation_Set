\section*{Problem 1}
Calculate the value of
\[
\cos^2 2190° + \frac{\sqrt{(-3)^2}}{i^{2020}} + \log_5 \frac{1}{125}
\]

\subsection*{Solution}
Let's evaluate each term separately.

First term: $\cos^2 2190°$
\begin{align}
2190° &= 6 \cdot 360° + 30° = 30°\\
\cos 2190° &= \cos 30° = \frac{\sqrt{3}}{2}\\
\cos^2 2190° &= \left(\frac{\sqrt{3}}{2}\right)^2 = \frac{3}{4}
\end{align}

Second term: $\frac{\sqrt{(-3)^2}}{i^{2020}}$
\begin{align}
\sqrt{(-3)^2} &= \sqrt{9} = 3\\
i^{2020} &= i^{4 \cdot 505} = (i^4)^{505} = 1^{505} = 1\\
\frac{\sqrt{(-3)^2}}{i^{2020}} &= \frac{3}{1} = 3
\end{align}

Third term: $\log_5 \frac{1}{125}$
\begin{align}
\frac{1}{125} &= \frac{1}{5^3} = 5^{-3}\\
\log_5 \frac{1}{125} &= \log_5 5^{-3} = -3
\end{align}

Therefore:
\[
\cos^2 2190° + \frac{\sqrt{(-3)^2}}{i^{2020}} + \log_5 \frac{1}{125} = \frac{3}{4} + 3 + (-3) = \frac{3}{4}
\]

\subsection*{Answer}
$\frac{3}{4}$ (option \textbf{A}).

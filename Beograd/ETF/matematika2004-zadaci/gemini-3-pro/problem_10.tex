\section*{Problem 10}
Ako je $\cos 2x = 1/2$ pri Äemu je $0 < x < \pi$, tada je $\sin 7x$ jednako:
\begin{itemize}
    \item[A)] $0$
    \item[B)] $\frac{\sqrt{3}}{2}$
    \item[C)] $1$
    \item[D)] $-1$
    \item[E)] $-\frac{1}{2}$
    \item[N)] Ne znam
\end{itemize}

\subsection*{Solution}
The equation $\cos 2x = 1/2$ has solutions for $2x$:
\[ 2x = \frac{\pi}{3} + 2k\pi \quad \text{or} \quad 2x = -\frac{\pi}{3} + 2k\pi. \]
Given $0 < x < \pi$, we have $0 < 2x < 2\pi$.
Possible values for $2x$ in this range are $\frac{\pi}{3}$ and $2\pi - \frac{\pi}{3} = \frac{5\pi}{3}$.
Thus, $x = \frac{\pi}{6}$ or $x = \frac{5\pi}{6}$.
We need to calculate $\sin 7x$.
Case 1: $x = \frac{\pi}{6}$.
\[ \sin 7x = \sin \frac{7\pi}{6} = \sin(\pi + \frac{\pi}{6}) = -\sin \frac{\pi}{6} = -\frac{1}{2}. \]
Case 2: $x = \frac{5\pi}{6}$.
\[ \sin 7x = \sin \frac{35\pi}{6} = \sin(6\pi - \frac{\pi}{6}) = \sin(-\frac{\pi}{6}) = -\frac{1}{2}. \]
In both cases, the result is $-\frac{1}{2}$.

\subsection*{Answer}
The value is $-\frac{1}{2}$, which corresponds to option \textbf{E}.

\section*{Problem 5}
Ako je $\frac{a}{A} = \frac{b}{B} = \frac{c}{C} = k$, tada je $\frac{4a-3b+c}{4A-3B+C}$ jednako:
\begin{itemize}
    \item[A)] $2k$
    \item[B)] $k$
    \item[C)] $\frac{2k}{3}$
    \item[D)] $-2k$
    \item[E)] $\frac{k}{2}$
    \item[N)] Ne znam
\end{itemize}

\subsection*{Solution}
From the given equalities, we can express $a, b, c$ in terms of $A, B, C$ and $k$:
\[ a = kA, \quad b = kB, \quad c = kC. \]
Substitute these into the numerator of the target expression:
\[ \frac{4a-3b+c}{4A-3B+C} = \frac{4(kA) - 3(kB) + (kC)}{4A - 3B + C} = \frac{k(4A - 3B + C)}{4A - 3B + C}. \]
Assuming $4A - 3B + C \neq 0$, we can cancel the term in the parentheses:
\[ \frac{k(4A - 3B + C)}{4A - 3B + C} = k. \]

\subsection*{Answer}
The value is $k$, which corresponds to option \textbf{B}.

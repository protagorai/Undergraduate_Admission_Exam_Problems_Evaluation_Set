\section*{Problem 10}
Ako je $\cos 2x = 1/2$ pri čemu je $0 < x < \pi$, tada je $\sin 7x$ jednako:

\subsection*{Solution}
Dato je $\cos 2x = \frac{1}{2}$ i $0 < x < \pi$.

Iz $\cos 2x = \frac{1}{2}$, imamo $2x = \frac{\pi}{3}$ ili $2x = \frac{5\pi}{3}$.

Pošto je $0 < x < \pi$, imamo $0 < 2x < 2\pi$.

Dakle, $2x = \frac{\pi}{3}$ ili $2x = \frac{5\pi}{3}$.

To znači $x = \frac{\pi}{6}$ ili $x = \frac{5\pi}{6}$.

Za $x = \frac{\pi}{6}$:
\[
\sin 7x = \sin\left(7 \cdot \frac{\pi}{6}\right) = \sin\left(\frac{7\pi}{6}\right) = -\sin\left(\frac{\pi}{6}\right) = -\frac{1}{2}
\]

Za $x = \frac{5\pi}{6}$:
\[
\sin 7x = \sin\left(7 \cdot \frac{5\pi}{6}\right) = \sin\left(\frac{35\pi}{6}\right) = \sin\left(\frac{35\pi}{6} - 6\pi\right) = \sin\left(-\frac{\pi}{6}\right) = -\frac{1}{2}
\]

U oba slučaja, $\sin 7x = -\frac{1}{2}$.

\subsection*{Answer}
$-\frac{1}{2}$ (option \textbf{E}).

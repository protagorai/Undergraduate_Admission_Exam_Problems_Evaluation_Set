\section*{Problem 8}
Ako je $\log_a x + \log_a b = 1, b > 0, b \neq 1, x \neq 1$, tada je $x$ jednako:

\subsection*{Solution}
Dato je $\log_a x + \log_a b = 1$.

Koristeći svojstvo logaritma $\log_a m + \log_a n = \log_a(mn)$:
\[
\log_a x + \log_a b = \log_a(xb) = 1
\]

Ovo znači:
\[
xb = a^1 = a
\]

Dakle:
\[
x = \frac{a}{b}
\]

\subsection*{Answer}
$\frac{a}{b}$ (option \textbf{A}).

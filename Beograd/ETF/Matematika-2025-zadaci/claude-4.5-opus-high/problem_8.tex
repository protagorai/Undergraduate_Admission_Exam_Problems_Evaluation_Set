\section*{Problem 8}
Zbir svih realnih rešenja jednačine $8^x - \frac{21}{4} \cdot 4^x + \frac{21}{4} \cdot 2^x - 1 = 0$ iznosi:

\subsection*{Solution}
Neka je $t = 2^x$. Tada:
\begin{itemize}
    \item $4^x = (2^2)^x = 2^{2x} = t^2$
    \item $8^x = (2^3)^x = 2^{3x} = t^3$
\end{itemize}

Jednačina postaje:
\[
t^3 - \frac{21}{4}t^2 + \frac{21}{4}t - 1 = 0
\]

Množimo sa 4:
\[
4t^3 - 21t^2 + 21t - 4 = 0
\]

Tražimo racionalne korene. Probamo $t = 1$:
\[
4 - 21 + 21 - 4 = 0 \checkmark
\]

Dakle $t = 1$ je koren, pa je $(t - 1)$ faktor.

Delimo polinom:
\[
4t^3 - 21t^2 + 21t - 4 = (t - 1)(4t^2 - 17t + 4)
\]

Proveravamo: $(t-1)(4t^2 - 17t + 4) = 4t^3 - 17t^2 + 4t - 4t^2 + 17t - 4 = 4t^3 - 21t^2 + 21t - 4$ \checkmark

Rešavamo $4t^2 - 17t + 4 = 0$:
\[
t = \frac{17 \pm \sqrt{289 - 64}}{8} = \frac{17 \pm \sqrt{225}}{8} = \frac{17 \pm 15}{8}
\]

\[
t_1 = \frac{17 + 15}{8} = \frac{32}{8} = 4
\]
\[
t_2 = \frac{17 - 15}{8} = \frac{2}{8} = \frac{1}{4}
\]

Dakle koreni su $t = 1$, $t = 4$, $t = \frac{1}{4}$.

Sada nalazimo $x$:
\begin{itemize}
    \item $2^x = 1 \Rightarrow x = 0$
    \item $2^x = 4 = 2^2 \Rightarrow x = 2$
    \item $2^x = \frac{1}{4} = 2^{-2} \Rightarrow x = -2$
\end{itemize}

Zbir svih realnih rešenja:
\[
0 + 2 + (-2) = 0
\]

\subsection*{Answer}
$0$ (opcija \textbf{B}).

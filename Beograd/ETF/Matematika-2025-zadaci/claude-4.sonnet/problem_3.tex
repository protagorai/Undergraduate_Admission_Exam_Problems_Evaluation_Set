\section*{Problem 3}
If $\log_a b = 6$ for $a,b \in (0,1) \cup (1,+\infty)$, find $\log_b(a^2b) + \log_{\sqrt{a}}\left(\frac{a}{\sqrt{b}}\right)$.

\subsection*{Solution}
Given: $\log_a b = 6$, which means $a^6 = b$.

Let's compute each term separately.

\textbf{First term:} $\log_b(a^2b)$
\[
\log_b(a^2b) = \log_b(a^2) + \log_b(b) = 2\log_b(a) + 1
\]

Since $\log_a b = 6$, we have $\log_b a = \frac{1}{\log_a b} = \frac{1}{6}$.

Therefore: $\log_b(a^2b) = 2 \cdot \frac{1}{6} + 1 = \frac{1}{3} + 1 = \frac{4}{3}$.

\textbf{Second term:} $\log_{\sqrt{a}}\left(\frac{a}{\sqrt{b}}\right)$

First, note that $\sqrt{a} = a^{1/2}$ and $\sqrt{b} = b^{1/2}$.

\[
\log_{\sqrt{a}}\left(\frac{a}{\sqrt{b}}\right) = \log_{a^{1/2}}\left(\frac{a}{b^{1/2}}\right)
\]

Using the change of base formula:
\[
\log_{a^{1/2}}\left(\frac{a}{b^{1/2}}\right) = \frac{\log_a\left(\frac{a}{b^{1/2}}\right)}{\log_a(a^{1/2})} = \frac{\log_a(a) - \log_a(b^{1/2})}{\frac{1}{2}} = \frac{1 - \frac{1}{2}\log_a(b)}{\frac{1}{2}}
\]

Since $\log_a b = 6$:
\[
= \frac{1 - \frac{1}{2} \cdot 6}{\frac{1}{2}} = \frac{1 - 3}{\frac{1}{2}} = \frac{-2}{\frac{1}{2}} = -4
\]

\textbf{Final answer:}
\[
\log_b(a^2b) + \log_{\sqrt{a}}\left(\frac{a}{\sqrt{b}}\right) = \frac{4}{3} + (-4) = \frac{4}{3} - 4 = \frac{4 - 12}{3} = -\frac{8}{3}
\]

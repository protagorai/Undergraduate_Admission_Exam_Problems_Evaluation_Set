\section*{Problem 17}
If polynomial $P_n(x)$ (where $n \geq 3$) when divided by $x$ gives remainder $3$, and when divided by $x^2 + 1$ gives remainder $-x + 2$, then $(P_n(x))^2 + P_n(x) + 1$ when divided by $x^3 + x$ gives remainder:

\subsection*{Solution}
From the given conditions:
1. $P_n(x) = x \cdot Q_1(x) + 3$ for some polynomial $Q_1(x)$
2. $P_n(x) = (x^2 + 1) \cdot Q_2(x) + (-x + 2)$ for some polynomial $Q_2(x)$

From condition 1: $P_n(0) = 3$

From condition 2, we can find:
- $P_n(i) = -i + 2$ (where $i^2 = -1$)
- $P_n(-i) = -(-i) + 2 = i + 2$

Now we need to find the remainder when $(P_n(x))^2 + P_n(x) + 1$ is divided by $x^3 + x = x(x^2 + 1)$.

Since $x^3 + x = x(x^2 + 1)$, the remainder must be a polynomial of degree at most 2. Let's call it $R(x) = ax^2 + bx + c$.

So: $(P_n(x))^2 + P_n(x) + 1 = (x^3 + x) \cdot Q(x) + ax^2 + bx + c$

To find $a$, $b$, and $c$, we evaluate at the roots of $x^3 + x = 0$, which are $x = 0, i, -i$.

At $x = 0$:
\[
(P_n(0))^2 + P_n(0) + 1 = 3^2 + 3 + 1 = 9 + 3 + 1 = 13
\]
So $R(0) = c = 13$.

At $x = i$:
\[
(P_n(i))^2 + P_n(i) + 1 = (-i + 2)^2 + (-i + 2) + 1
\]
\[
= (i^2 - 4i + 4) + (-i + 2) + 1 = (-1 - 4i + 4) + (-i + 2) + 1
\]
\[
= 3 - 4i - i + 2 + 1 = 6 - 5i
\]

So $R(i) = ai^2 + bi + c = -a + bi + 13 = 6 - 5i$.
This gives us: $-a + 13 = 6$ and $b = -5$.
Therefore: $a = 7$ and $b = -5$.

Let's verify with $x = -i$:
\[
(P_n(-i))^2 + P_n(-i) + 1 = (i + 2)^2 + (i + 2) + 1
\]
\[
= (i^2 + 4i + 4) + (i + 2) + 1 = (-1 + 4i + 4) + (i + 2) + 1
\]
\[
= 3 + 4i + i + 2 + 1 = 6 + 5i
\]

Therefore, the remainder is $R(x) = 7x^2 - 5x + 13$.

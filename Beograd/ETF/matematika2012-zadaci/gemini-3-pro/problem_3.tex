\section*{Problem 3}
Ako je $f(x) = x^3 - 3x$ i $g(x) = \sin \frac{\pi}{12}x$, tada je $f(g(2))$ jednako:

\subsection*{Solution}
First calculate $g(2)$:
\[
g(2) = \sin\left(\frac{\pi}{12} \cdot 2\right) = \sin\left(\frac{\pi}{6}\right) = \frac{1}{2}
\]
Now calculate $f(g(2)) = f(1/2)$:
\[
f(1/2) = \left(\frac{1}{2}\right)^3 - 3\left(\frac{1}{2}\right) = \frac{1}{8} - \frac{3}{2} = \frac{1 - 12}{8} = -\frac{11}{8}
\]

\subsection*{Answer}
(D) $-\frac{11}{8}$

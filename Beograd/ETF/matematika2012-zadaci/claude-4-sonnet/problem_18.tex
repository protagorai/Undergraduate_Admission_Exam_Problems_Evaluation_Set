\section*{Problem 18}
Skup svih realnih rešenja nejednačine $\frac{|1 - x|}{1 - |x|} < \frac{1 + |x|}{|1 + x|}$ je oblik (za neke realne brojeve $a$ i $b$ takve da je $0 < a < b < +\infty$):

\subsection*{Solution}
Analiziramo nejednačinu $\frac{|1 - x|}{1 - |x|} < \frac{1 + |x|}{|1 + x|}$.

Prvo određujemo oblast definisanosti:
- $1 - |x| \neq 0 \Rightarrow |x| \neq 1 \Rightarrow x \neq \pm 1$
- $|1 + x| \neq 0 \Rightarrow x \neq -1$

Dakle, $x \neq \pm 1$.

Razmotrimo različite slučajeve:

**Slučaj 1: $x > 1$**
- $|x| = x$, $|1 - x| = x - 1$, $|1 + x| = 1 + x$
- Nejednačina postaje: $\frac{x - 1}{1 - x} < \frac{1 + x}{1 + x}$
- $\frac{x - 1}{1 - x} < 1$
- $\frac{-(1 - x)}{1 - x} < 1$
- $-1 < 1$ (uvek tačno)

**Slučaj 2: $0 < x < 1$**
- $|x| = x$, $|1 - x| = 1 - x$, $|1 + x| = 1 + x$
- Nejednačina postaje: $\frac{1 - x}{1 - x} < \frac{1 + x}{1 + x}$
- $1 < 1$ (nikad tačno)

**Slučaj 3: $-1 < x < 0$**
- $|x| = -x$, $|1 - x| = 1 - x$, $|1 + x| = 1 + x$
- Nejednačina postaje: $\frac{1 - x}{1 - (-x)} < \frac{1 + (-x)}{1 + x}$
- $\frac{1 - x}{1 + x} < \frac{1 - x}{1 + x}$ (nikad tačno)

**Slučaj 4: $x < -1$**
- $|x| = -x$, $|1 - x| = 1 - x$, $|1 + x| = -(1 + x)$
- Nejednačina postaje: $\frac{1 - x}{1 + x} < \frac{1 - x}{-(1 + x)}$
- $\frac{1 - x}{1 + x} < -\frac{1 - x}{1 + x}$

Pošto je $x < -1$, imamo $1 + x < 0$ i $1 - x > 0$.
Dakle, $\frac{1 - x}{1 + x} < 0$.

Nejednačina postaje: negativan broj $<$ pozitivan broj, što je tačno.

Dakle, rešenje je $x \in (-\infty, -1) \cup (1, +\infty)$.

\subsection*{Answer}
$(-\infty, -a) \cup (a, +\infty)$ gde je $a = 1$ (opcija \textbf{C}).

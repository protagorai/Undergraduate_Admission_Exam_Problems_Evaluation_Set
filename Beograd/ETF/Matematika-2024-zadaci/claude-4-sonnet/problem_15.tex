\section*{Problem 15}
Broj rešenja jednačine $\frac{\cos 3x + \cos 4x + \cos 5x}{\sin x - \sqrt{3}\cos x} = 0$ na segmentu $[0, \pi]$ iznosi:

\subsection*{Solution}
Jednačina je zadovoljena kada je brojnik jednak nuli, a imenilac različit od nule.

Brojnik: $\cos 3x + \cos 4x + \cos 5x = 0$

Koristimo formule za zbir kosinusa:
\[\cos 3x + \cos 5x = 2\cos\left(\frac{3x+5x}{2}\right)\cos\left(\frac{5x-3x}{2}\right) = 2\cos(4x)\cos(x)\]

Dakle:
\[\cos 3x + \cos 4x + \cos 5x = 2\cos(4x)\cos(x) + \cos(4x) = \cos(4x)(2\cos(x) + 1) = 0\]

Ovo se dešava kada:
1. $\cos(4x) = 0 \Rightarrow 4x = \frac{\pi}{2} + k\pi \Rightarrow x = \frac{\pi}{8} + \frac{k\pi}{4}$
2. $2\cos(x) + 1 = 0 \Rightarrow \cos(x) = -\frac{1}{2} \Rightarrow x = \frac{2\pi}{3}, \frac{4\pi}{3}$

Na segmentu $[0, \pi]$:
- Za $\cos(4x) = 0$: $x = \frac{\pi}{8}, \frac{3\pi}{8}, \frac{5\pi}{8}, \frac{7\pi}{8}$
- Za $\cos(x) = -\frac{1}{2}$: $x = \frac{2\pi}{3}$

Imenilac: $\sin x - \sqrt{3}\cos x = 0 \Rightarrow \tan x = \sqrt{3} \Rightarrow x = \frac{\pi}{3}, \frac{4\pi}{3}$

Na segmentu $[0, \pi]$, imenilac je nula za $x = \frac{\pi}{3}$.

Proveravamo da li se neko od rešenja brojnika poklapa sa $x = \frac{\pi}{3}$:
- $\frac{\pi}{3} \neq \frac{\pi}{8}, \frac{3\pi}{8}, \frac{5\pi}{8}, \frac{7\pi}{8}, \frac{2\pi}{3}$

Dakle, sva rešenja brojnika su validna: 5 rešenja.

\subsection*{Answer}
$5$ (opcija \textbf{C}).

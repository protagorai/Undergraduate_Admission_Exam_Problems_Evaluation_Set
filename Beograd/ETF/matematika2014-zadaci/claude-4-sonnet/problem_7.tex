\section*{Problem 7}
Ako je $a = -0,3$ koja od sledećih relacija je tačna?

(A) $a < a^2 < a^3$ \quad (B) $a < a^3 < a^2$ \quad (C) $a^2 < a < a^3$ \quad (D) $a^2 < a^3 < a$ \quad (E) $a^3 < a < a^2$ \quad (N) Ne znam

\subsection*{Solution}
Imamo $a = -0,3 = -\frac{3}{10}$.

Računamo stepene:
\[
a = -0,3
\]
\[
a^2 = (-0,3)^2 = 0,09
\]
\[
a^3 = (-0,3)^3 = -0,027
\]

Poređenjem brojeva:
- $a = -0,3 < 0$
- $a^2 = 0,09 > 0$
- $a^3 = -0,027 < 0$

Među negativnim brojevima: $-0,3 < -0,027$, tj. $a < a^3$.

Dakle imamo: $a < a^3 < 0 < a^2$, što znači $a < a^3 < a^2$.

\subsection*{Answer}
$a < a^3 < a^2$ (option \textbf{B}).

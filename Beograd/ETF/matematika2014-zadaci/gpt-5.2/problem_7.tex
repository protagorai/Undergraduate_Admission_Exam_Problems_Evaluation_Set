\section*{Problem 7}
Ako je $a=-0{,}3$ koja od sledećih relacija je tačna?
\[
\text{(A) } a<a^2<a^3\quad
\text{(B) } a<a^3<a^2\quad
\text{(C) } a^2<a<a^3\quad
\text{(D) } a^2<a^3<a\quad
\text{(E) } a^3<a<a^2.
\]

\subsection*{Solution}
Za $a=-0.3$ važi:
\[
a=-0.3,\qquad a^2=0.09>0,\qquad a^3=-0.027.
\]
Dakle je redosled
\[
a=-0.3 < a^3=-0.027 < a^2=0.09,
\]
što odgovara opciji (B).

\subsection*{Answer}
$a<a^3<a^2$ (option \textbf{B}).

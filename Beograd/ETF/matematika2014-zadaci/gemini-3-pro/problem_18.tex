\section*{Problem 18}
Neka je $S$ skup svih realnih rešenja nejednačine $\text{tg}x(1-\text{tg}^2x)(1-3\text{tg}^2x)(1+\text{tg}2x \cdot \text{tg}3x) > 0$ i neka je $S_1 \subset S$. Tada skup $S_1$ može biti:
\begin{enumerate}
    \item[(A)] $\left(-\frac{\pi}{2}, \frac{\pi}{2}\right)$
    \item[(B)] $\left(\frac{\pi}{3}, \frac{\pi}{2}\right)$
    \item[(C)] $\left(\frac{3\pi}{4}, \pi\right)$
    \item[(D)] $\left(\frac{7\pi}{6}, \frac{3\pi}{2}\right)$
    \item[(E)] $\left(\frac{\pi}{2}, \frac{5\pi}{6}\right)$
    \item[(N)] Ne znam
\end{enumerate}

\subsection*{Solution}
Let's analyze the factors.
1. $\tan x$
2. $1-\tan^2 x$
3. $1-3\tan^2 x$
4. $1+\tan 2x \tan 3x$

Simplify the fourth factor:
\[
1+\tan 2x \tan 3x = 1 + \frac{\sin 2x \sin 3x}{\cos 2x \cos 3x} = \frac{\cos 2x \cos 3x + \sin 2x \sin 3x}{\cos 2x \cos 3x} = \frac{\cos(3x-2x)}{\cos 2x \cos 3x} = \frac{\cos x}{\cos 2x \cos 3x}
\]
The inequality becomes:
\[
\tan x (1-\tan^2 x) (1-3\tan^2 x) \frac{\cos x}{\cos 2x \cos 3x} > 0
\]
Using identities $\tan 2x = \frac{2\tan x}{1-\tan^2 x}$ and $\tan 3x = \frac{3\tan x - \tan^3 x}{1-3\tan^2 x} = \tan x \frac{3-\tan^2 x}{1-3\tan^2 x}$:
We can substitute terms, but let's just evaluate signs in the interval for option (B): $(\frac{\pi}{3}, \frac{\pi}{2})$.
Interval $(\frac{\pi}{3}, \frac{\pi}{2})$ corresponds to $(60^\circ, 90^\circ)$.
In this interval:
- $\tan x > \sqrt{3} > 0$ (Positive)
- $1-\tan^2 x < 1-3 = -2$ (Negative)
- $1-3\tan^2 x < 1-9 = -8$ (Negative)
- $\cos x > 0$ (Positive)
- $2x \in (120^\circ, 180^\circ) \implies \cos 2x < 0$ (Negative)
- $3x \in (180^\circ, 270^\circ) \implies \cos 3x < 0$ (Negative)

The fourth factor term $\frac{\cos x}{\cos 2x \cos 3x}$: $\frac{(+)}{(-)(-)} = (+)$.
So the product of signs is:
$(+) \cdot (-) \cdot (-) \cdot (+) = (+)$
The inequality holds. The product is positive.
So $(\frac{\pi}{3}, \frac{\pi}{2}) \subset S$.

\subsection*{Answer}
$\left(\frac{\pi}{3}, \frac{\pi}{2}\right)$ (option \textbf{(B)}).

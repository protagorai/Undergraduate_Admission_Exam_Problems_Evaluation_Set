\section*{Problem 7}
Ako je $a = -0,3$ koja od sledećih relacija je tačna?
\begin{enumerate}
    \item[(A)] $a < a^2 < a^3$
    \item[(B)] $a < a^3 < a^2$
    \item[(C)] $a^2 < a < a^3$
    \item[(D)] $a^2 < a^3 < a$
    \item[(E)] $a^3 < a < a^2$
    \item[(N)] Ne znam
\end{enumerate}

\subsection*{Solution}
Given $a = -0.3$.
Calculate powers:
\[
a^2 = (-0.3)^2 = 0.09
\]
\[
a^3 = (-0.3)^3 = -0.027
\]
Comparing the values:
$a = -0.3$
$a^3 = -0.027$
$a^2 = 0.09$

Since $-0.3 < -0.027$, we have $a < a^3$.
Since $-0.027 < 0.09$, we have $a^3 < a^2$.
So, $a < a^3 < a^2$.

\subsection*{Answer}
$a < a^3 < a^2$ (option \textbf{(B)}).

\section*{Problem 12}
Krug sadrži tri tačke čije su koordinate $(0,6)$, $(0,10)$ i $(8,0)$. Apscisa druge tačke u kojoj dati krug seče $x$-osu, jednaka je:

(A) $7$ \quad (B) $7, 25$ \quad (C) $7, 5$ \quad (D) $7, 75$ \quad (E) $9$ \quad (N) Ne znam

\subsection*{Solution}
The general equation of a circle is:
\[
x^2 + y^2 + Dx + Ey + F = 0
\]

Substituting the three given points:

For $(0, 6)$: $0 + 36 + 0 + 6E + F = 0 \implies 6E + F = -36$ ... (1)

For $(0, 10)$: $0 + 100 + 0 + 10E + F = 0 \implies 10E + F = -100$ ... (2)

For $(8, 0)$: $64 + 0 + 8D + 0 + F = 0 \implies 8D + F = -64$ ... (3)

From (2) - (1): $4E = -64 \implies E = -16$

From (1): $6(-16) + F = -36 \implies F = -36 + 96 = 60$

From (3): $8D + 60 = -64 \implies 8D = -124 \implies D = -\frac{124}{8} = -15.5$

The circle equation is:
\[
x^2 + y^2 - 15.5x - 16y + 60 = 0
\]

To find where the circle intersects the $x$-axis, set $y = 0$:
\[
x^2 - 15.5x + 60 = 0
\]

Using the quadratic formula:
\[
x = \frac{15.5 \pm \sqrt{240.25 - 240}}{2} = \frac{15.5 \pm \sqrt{0.25}}{2} = \frac{15.5 \pm 0.5}{2}
\]

So $x_1 = \frac{16}{2} = 8$ (which is the given point) and $x_2 = \frac{15}{2} = 7.5$.

\subsection*{Answer}
$7.5$ (option \textbf{C}).

\section*{Problem 20}
All real solutions of the equation $\frac{x + \sqrt{3}}{\sqrt{x} + \sqrt{x + \sqrt{3}}} + \frac{x - \sqrt{3}}{\sqrt{x} - \sqrt{x - \sqrt{3}}} = \sqrt{x}$ are in the set:

\subsection*{Solution}
Let's denote $a = \sqrt{x}$, $b = \sqrt{x + \sqrt{3}}$, and $c = \sqrt{x - \sqrt{3}}$.

Note that we need $x \geq 0$, $x + \sqrt{3} \geq 0$ (always true for $x \geq 0$), and $x - \sqrt{3} \geq 0$, so $x \geq \sqrt{3}$.

The equation becomes:
\[\frac{x + \sqrt{3}}{a + b} + \frac{x - \sqrt{3}}{a - c} = a\]

Since $b^2 = x + \sqrt{3}$ and $c^2 = x - \sqrt{3}$, we have:
\[\frac{b^2}{a + b} + \frac{c^2}{a - c} = a\]

Let's rationalize the denominators:
\[\frac{b^2}{a + b} = \frac{b^2(a - b)}{(a + b)(a - b)} = \frac{b^2(a - b)}{a^2 - b^2}\]

Since $a^2 = x$ and $b^2 = x + \sqrt{3}$, we have $a^2 - b^2 = x - (x + \sqrt{3}) = -\sqrt{3}$.

So:
\[\frac{b^2}{a + b} = \frac{b^2(a - b)}{-\sqrt{3}} = \frac{(x + \sqrt{3})(a - b)}{-\sqrt{3}}\]

Similarly:
\[\frac{c^2}{a - c} = \frac{c^2(a + c)}{(a - c)(a + c)} = \frac{c^2(a + c)}{a^2 - c^2}\]

Since $c^2 = x - \sqrt{3}$, we have $a^2 - c^2 = x - (x - \sqrt{3}) = \sqrt{3}$.

So:
\[\frac{c^2}{a - c} = \frac{(x - \sqrt{3})(a + c)}{\sqrt{3}}\]

The equation becomes:
\[\frac{(x + \sqrt{3})(a - b)}{-\sqrt{3}} + \frac{(x - \sqrt{3})(a + c)}{\sqrt{3}} = a\]

Multiplying by $\sqrt{3}$:
\[-(x + \sqrt{3})(a - b) + (x - \sqrt{3})(a + c) = a\sqrt{3}\]

Let's expand:
\[-(x + \sqrt{3})a + (x + \sqrt{3})b + (x - \sqrt{3})a + (x - \sqrt{3})c = a\sqrt{3}\]
\[a[-(x + \sqrt{3}) + (x - \sqrt{3})] + (x + \sqrt{3})b + (x - \sqrt{3})c = a\sqrt{3}\]
\[a(-2\sqrt{3}) + (x + \sqrt{3})b + (x - \sqrt{3})c = a\sqrt{3}\]
\[-2a\sqrt{3} + (x + \sqrt{3})b + (x - \sqrt{3})c = a\sqrt{3}\]
\[(x + \sqrt{3})b + (x - \sqrt{3})c = 3a\sqrt{3}\]

Substituting $b = \sqrt{x + \sqrt{3}}$ and $c = \sqrt{x - \sqrt{3}}$:
\[(x + \sqrt{3})\sqrt{x + \sqrt{3}} + (x - \sqrt{3})\sqrt{x - \sqrt{3}} = 3\sqrt{x}\sqrt{3}\]

This is getting quite complex. Let me try specific values.

Let's try $x = 3$:
- $a = \sqrt{3}$
- $b = \sqrt{3 + \sqrt{3}}$  
- $c = \sqrt{3 - \sqrt{3}}$

The original equation becomes:
\[\frac{3 + \sqrt{3}}{\sqrt{3} + \sqrt{3 + \sqrt{3}}} + \frac{3 - \sqrt{3}}{\sqrt{3} - \sqrt{3 - \sqrt{3}}} = \sqrt{3}\]

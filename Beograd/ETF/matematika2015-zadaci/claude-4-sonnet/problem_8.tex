\section*{Problem 8}
Given that $\cos\left(x - \frac{3\pi}{2}\right) = -\frac{4}{5}$ and $\frac{\pi}{2} < x < \pi$, the value of the expression $\frac{x}{2}\cos\frac{5x}{2}$ is:

\subsection*{Solution}
First, let's simplify $\cos\left(x - \frac{3\pi}{2}\right)$.

Using the identity $\cos(A - B) = \cos A \cos B + \sin A \sin B$:
\[\cos\left(x - \frac{3\pi}{2}\right) = \cos x \cos\left(\frac{3\pi}{2}\right) + \sin x \sin\left(\frac{3\pi}{2}\right)\]

Since $\cos\left(\frac{3\pi}{2}\right) = 0$ and $\sin\left(\frac{3\pi}{2}\right) = -1$:
\[\cos\left(x - \frac{3\pi}{2}\right) = \cos x \cdot 0 + \sin x \cdot (-1) = -\sin x\]

Therefore: $-\sin x = -\frac{4}{5}$, which gives us $\sin x = \frac{4}{5}$.

Since $\frac{\pi}{2} < x < \pi$ (second quadrant), we have $\cos x < 0$.

Using the Pythagorean identity: $\cos^2 x + \sin^2 x = 1$
\[\cos^2 x = 1 - \sin^2 x = 1 - \left(\frac{4}{5}\right)^2 = 1 - \frac{16}{25} = \frac{9}{25}\]

Since $\cos x < 0$ in the second quadrant: $\cos x = -\frac{3}{5}$.

Now we need to find $\frac{x}{2}\cos\frac{5x}{2}$.

From $\sin x = \frac{4}{5}$ and $\cos x = -\frac{3}{5}$, we can find $x$.
We have $\tan x = \frac{\sin x}{\cos x} = \frac{4/5}{-3/5} = -\frac{4}{3}$.

Since $x$ is in the second quadrant and $\tan x = -\frac{4}{3}$, we have:
$x = \pi - \arctan\left(\frac{4}{3}\right)$

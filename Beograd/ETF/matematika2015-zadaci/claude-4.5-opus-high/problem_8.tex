\section*{Problem 8}
ZnajuÄ‡i da je $\cos\left(x - \dfrac{3\pi}{2}\right) = -\dfrac{4}{5}$ i $\dfrac{\pi}{2} < x < \pi$, tada je vrednost izraza $\sin\dfrac{x}{2}\cos\dfrac{5x}{2}$ jednaka:

\subsection*{Solution}
First, simplify $\cos\left(x - \dfrac{3\pi}{2}\right)$:
\[
\cos\left(x - \frac{3\pi}{2}\right) = \cos x \cos\frac{3\pi}{2} + \sin x \sin\frac{3\pi}{2} = \cos x \cdot 0 + \sin x \cdot (-1) = -\sin x
\]

So $-\sin x = -\dfrac{4}{5}$, which means $\sin x = \dfrac{4}{5}$.

Since $\dfrac{\pi}{2} < x < \pi$, we have $\cos x < 0$:
\[
\cos x = -\sqrt{1 - \sin^2 x} = -\sqrt{1 - \frac{16}{25}} = -\frac{3}{5}
\]

Now we use the product-to-sum formula:
\[
\sin\frac{x}{2}\cos\frac{5x}{2} = \frac{1}{2}\left[\sin\left(\frac{x}{2} + \frac{5x}{2}\right) + \sin\left(\frac{x}{2} - \frac{5x}{2}\right)\right]
\]
\[
= \frac{1}{2}\left[\sin 3x + \sin(-2x)\right] = \frac{1}{2}\left[\sin 3x - \sin 2x\right]
\]

Calculate $\sin 2x$ and $\sin 3x$:
\[
\sin 2x = 2\sin x \cos x = 2 \cdot \frac{4}{5} \cdot \left(-\frac{3}{5}\right) = -\frac{24}{25}
\]

\[
\cos 2x = \cos^2 x - \sin^2 x = \frac{9}{25} - \frac{16}{25} = -\frac{7}{25}
\]

\[
\sin 3x = \sin(2x + x) = \sin 2x \cos x + \cos 2x \sin x
\]
\[
= \left(-\frac{24}{25}\right)\left(-\frac{3}{5}\right) + \left(-\frac{7}{25}\right)\left(\frac{4}{5}\right) = \frac{72}{125} - \frac{28}{125} = \frac{44}{125}
\]

Therefore:
\[
\sin\frac{x}{2}\cos\frac{5x}{2} = \frac{1}{2}\left[\frac{44}{125} - \left(-\frac{24}{25}\right)\right] = \frac{1}{2}\left[\frac{44}{125} + \frac{120}{125}\right] = \frac{1}{2} \cdot \frac{164}{125} = \frac{82}{125}
\]

\subsection*{Answer}
$\dfrac{82}{125}$ (option \textbf{B}).

\section*{Problem 14}
Data je aritmetička progresija $a_1, a_2, \ldots$ čija je razlika $d = 1$, a zbir prvih 98 članova $a_1 + a_2 + \cdots + a_{98} = 137$. Tada je zbir $a_2 + a_4 + a_6 + \cdots + a_{98}$ jednak:

\subsection*{Solution}
For an arithmetic progression with first term $a_1$ and common difference $d = 1$:
\[
a_n = a_1 + (n-1)
\]

Sum of first 98 terms:
\[
S_{98} = \frac{98}{2}(2a_1 + 97 \cdot 1) = 49(2a_1 + 97) = 137
\]
\[
2a_1 + 97 = \frac{137}{49}
\]
\[
a_1 = \frac{137 - 49 \cdot 97}{98} = \frac{137 - 4753}{98} = \frac{-4616}{98} = -47.1...
\]

Actually, let me recalculate. We need:
\[
49(2a_1 + 97) = 137
\]

This gives a non-integer $a_1$, which is fine for this problem.

Now, we need $a_2 + a_4 + a_6 + \cdots + a_{98}$ (49 terms with even indices).

\[
a_{2k} = a_1 + (2k - 1) \text{ for } k = 1, 2, \ldots, 49
\]

Sum:
\[
\sum_{k=1}^{49} a_{2k} = \sum_{k=1}^{49} (a_1 + 2k - 1) = 49a_1 + 2\sum_{k=1}^{49}k - 49
\]
\[
= 49a_1 + 2 \cdot \frac{49 \cdot 50}{2} - 49 = 49a_1 + 2450 - 49 = 49a_1 + 2401
\]

From $S_{98} = 137$:
\[
S_{98} = 98a_1 + \sum_{k=0}^{97}k = 98a_1 + \frac{97 \cdot 98}{2} = 98a_1 + 4753 = 137
\]
\[
98a_1 = 137 - 4753 = -4616
\]
\[
a_1 = -\frac{4616}{98} = -47.1\overline{02}
\]

So $49a_1 = -\frac{4616}{2} = -2308$.

Sum of even-indexed terms:
\[
49a_1 + 2401 = -2308 + 2401 = 93
\]

\subsection*{Answer}
$93$ (option \textbf{B}).

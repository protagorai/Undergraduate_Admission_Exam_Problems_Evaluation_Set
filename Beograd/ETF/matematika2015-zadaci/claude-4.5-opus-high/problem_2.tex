\section*{Problem 2}
Najkraće rastojanje između pravih $\sqrt{2}x + y = 1$ i $2x + \sqrt{2}y = 3\sqrt{2}$ jednako je:

\subsection*{Solution}
First, let's rewrite the lines in standard form to check if they are parallel.

Line 1: $\sqrt{2}x + y = 1$

Line 2: $2x + \sqrt{2}y = 3\sqrt{2}$

Dividing Line 2 by $\sqrt{2}$:
\[
\sqrt{2}x + y = 3
\]

So both lines have the same direction vector, meaning they are parallel!

Line 1: $\sqrt{2}x + y = 1$
Line 2: $\sqrt{2}x + y = 3$

The distance between two parallel lines $ax + by = c_1$ and $ax + by = c_2$ is:
\[
d = \frac{|c_2 - c_1|}{\sqrt{a^2 + b^2}}
\]

Here $a = \sqrt{2}$, $b = 1$, $c_1 = 1$, $c_2 = 3$.

\[
d = \frac{|3 - 1|}{\sqrt{(\sqrt{2})^2 + 1^2}} = \frac{2}{\sqrt{2 + 1}} = \frac{2}{\sqrt{3}} = \frac{2\sqrt{3}}{3}
\]

\subsection*{Answer}
$\dfrac{2}{3}\sqrt{3}$ (option \textbf{D}).

\section*{Problem 4 - Solution}

\textbf{Problem:} Let $P(x) = ax^2 + bx + c$. If $P(1) = 1$, $P(0) = 2$, $P(-1) = 7$, find $a^2 + b^2 + c^2$.

\textbf{Solution:}

Set up the system of equations:

From $P(0) = c = 2$:
\[
c = 2
\]

From $P(1) = a + b + c = 1$:
\[
a + b + 2 = 1 \implies a + b = -1 \quad (1)
\]

From $P(-1) = a - b + c = 7$:
\[
a - b + 2 = 7 \implies a - b = 5 \quad (2)
\]

Adding equations (1) and (2):
\[
2a = 4 \implies a = 2
\]

From equation (1):
\[
b = -1 - a = -1 - 2 = -3
\]

Therefore: $a = 2$, $b = -3$, $c = 2$

Calculate $a^2 + b^2 + c^2$:
\[
a^2 + b^2 + c^2 = 2^2 + (-3)^2 + 2^2 = 4 + 9 + 4 = 17
\]
